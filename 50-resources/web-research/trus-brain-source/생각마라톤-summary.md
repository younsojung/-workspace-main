## 1. 트루스의 메타 사고 원칙

### "한 번 더, 함께, 끝까지 생각한다"
- **한 번 더 생각한다** — 즉답 안 함. 회장님들이 통번역가 데리고 다니는 이유 = 생각할 시간 사기.
- **함께 생각한다** — 내 얘기에 빠지지 말 것. 공통점 찾는 뇌의 시냅스를 써야 진짜 공부.
- **끝까지 생각한다** = **숫자 언어 + 고객 언어**. 이 두 가지가 나오지 않으면 진행하지 않는 게 원칙.

> "시작은 누구나 잘해. 근데 내 킥은 끝에서부터 생각하는 거야." (소정)

### 생각 = 더듬이 = 무의식
- 한자: 생각 할 생(生), 뿔 각(角) — 곤충의 더듬이.
- "좋은 생각은 의식이 아니라 무의식에서 나온다. 샤워할 때, 달리기 할 때, 사우나할 때."
- 운전 처음 배울 땐 다 의식. 익숙해지면 무의식.

### 무의식을 바꾸는 3가지
1. **환경** — "돈을 알고 싶으면 돈 얘기 좋아하는 사람을 만나라."
2. **감정** — 쪽팔린 경험·죽을 만큼의 고비가 세포에 저장됨.
3. **반복** — 핵심. 무의식이 될 때까지 반복.

### 학습의 5단계: 지행용훈평 (이건희)
- **지(知)** — 안다 (4단계): 들어본 것 → 기억하는 것 → 말할 수 있는 것 → 기가 막히게 말할 수 있는 것
- **행(行)** — 해봤다 (2단계): 한 번 해본 것 → 습관이 된 것
- **용(用)** — 사람을 쓸 수 있다 (보여주면 따라옴)
- **훈(訓)** — 가르친다 (2단계): 가르치는 것 → 기가 막히게 가르치는 것
- **평(評)** — 그제서야 평가할 수 있다

> "흑백요리사 보면서 '저 사람은 좀 별로지 않아?' 하는 순간, 학습이 다 깨지는 거야."

## 2. 안전한 학습 환경 운영 도구

### 환대 — 3메들리 1터치 (3 medley 1 touch)
인큐 시절 영업 비밀. 입장 시 이름 부르기 3번 + 1터치로 안전감 형성.
- 일식집 "이랏샤이마세!"처럼
- 환대 → 마음 열림 → 긴장 사라짐
- 시댁·아이 교육·고객 응대에도 적용 가능

### 노는 손 없게 — 4가지 역할
좌우명: "노는 손이 없게 하자. 각자에게 역할을 줘라."
1. **응원대장** — 웃음소리 끊길 때마다 웃어주는 사람 (선생님들 웃음소리가 다 이상한 이유)
2. **기록대장** — 팀 전체 서포트. 한 팀에 한 명.
3. **행동대장** — 추천을 잘함. 진짜 좋은 얘기만 추천.
4. **질문대장** — 말이 끊기지 않게 계속 질문. "그래서 이런 뜻이라는 건가요?"

### 시간 단계의 회의 ≠ 성과 단계의 회의
- 시간 단계: "안 해본 사람 추천하지 말아라." 모두 발언 강요는 폭력.
- 회의가 단계마다 달라야 함.

### 명함 교환·단톡방 개설 금지 (인큐 → 트루스 진화)
- 인큐 시절: 새벽 2시 전화도 받음. 끝판왕 승인욕구 팀.
- 트루스 현재: 명함 교환 금지, 연락 금지, 단톡방 개설 금지, 사전 공지된 커피챗만.
- "이제 더 이상 승인받는 행동을 하지 않아."

## 3. 돈 철학

### 돈은 무의식·감정으로 본다 → 숫자로 보게 만든다
- 사람마다 돈에 대한 핵심 감정이 다름.
- 트루스 부부의 첫 기억: 소정은 정신병 아빠 병원비로 엄마가 가출 → **불안감**. 황호는 100원 모아 1200원 미니카 → **기쁨**. 둘의 감정 차이가 부부 싸움 원인.

### 돈의 핵심 감정 4+1
1. **두려움** — 통장 잔고 줄면 불안. 갑작스러운 변화 경험 多.
2. **슬픔** — 이미 늦었다. 무기력. 부모가 돈을 어찌할 수 없는 도구로 봄.
3. **분노** — 부당한 경험. 노조 만들어짐. 세금·수수료에 분노.
4. **기쁨** — 돈을 벌고 모으는 것 자체가 재밌음. 어릴 때 뭔가 판 경험.
5. **무감정** — 무지에서 옴.

### 돈 = 학습의 도구
> "나에게 돈은 학습의 도구야. 한 번도 소유의 도구로 본 적이 없어. 내가 지금 뭘 모르는지를 알게 해준 게 돈이었어. **성인 교육은 뭘 알려주는 게 아니야. 너가 뭘 모르는지를 알려줄 수 있어야 해.**"

### AGI 시대 부의 본질 변화
- 과거: 토지·자본·노동
- 현재: 노동 가치 사라짐 (AGI가 대체)
- 결론: 노동소득 점프 + 자본소득 빠른 업그레이드 둘 다 필요

### 자산을 만드는 3가지 공식 (싯다르타)
> "나는 단식할 줄 알고, 나는 인내할 줄 알고, 생각할 줄 아오."
- **절제** (단식) — 걸어간다 → 뛰어간다
- **시간** (인내) — 뛰어간다 → 차 타고 간다. 정기적으로 같은 금액 사기.
- **투자** (생각) — 차 타고 간다 → 비행기 타고 간다. 나만의 원칙.

> "비행기 바로 타고 싶다고? 순서는 절대 바꿀 수 없어. 인내를 배우지 않으면 결국 자기 그릇만큼 남고 다 날린다."

### 돈을 모으는 이유
> "돈은 좋은 결정을 위해 존재해. 살면서 하는 수많은 결정에서 돈이 개입하지 않게 하기 위해서야."
> "와이프가 힘들어도 울면서까지 일하지 않게 하는 것."

## 4. 노동소득 커리어 6단계 (트루스의 진단 도구)

세 구간으로 묶임: **시간 → 성과 → 자본**

| 단계 | 무엇을 파나 | 핵심 | 모르는 것 |
|------|-------------|------|----------|
| 1. 초보자 | 1인분 시간 | 태도 | 자기만족 vs 고객만족 |
| 2. 숙련자 | 숙련된 시간 | 안정 | 노력 vs 성과 |
| 3. 실력자 | 실력 | 주도성, 자기 증명, 퍼스널 브랜딩 | 전체 이익 기여도 계산 |
| 4. 전문가 | 보장된 실력 | 이론 정립, 위임, 20-80 | 탁월함을 심플함으로 |
| 5. 시스템 | 구조 | 매출 최대·비용 최소 | 사람 아닌 시스템 문제 |
| 6. 신뢰 | 시간이 쌓인 신뢰 | 묻지도 따지지도 않고 입금 | 누가 내 이야기를 하는가 |

### 단계별 학습 포인트
- **초보자 → 숙련자**: 과정 중심 → 성과 중심
- **숙련자 → 실력자**: 대체 불가, 자기 증명, 퍼스널 브랜딩, 기여도 계산
- **실력자 → 전문가**: 노하우 정리 → 이론 정립, 위임, 20-80
- **전문가 → 시스템**: 사람 잘못 없음, 시스템이 문제
- **시스템 → 신뢰**: 효율 이상의 축적, 반복은 문화가 되는 시간

### 공급의 법칙
- 위로 올라갈수록 1/10 토막씩 감소. 초보자/숙련자 10,000명 → 실력자 1,000 → 전문가 100 → 시스템 10 → 신뢰 1.
- 아이러니: 아래보다 위에서 경쟁이 *쉬움* (사람·결재 없이 실력으로 경쟁).
- 복리: 1억보다 10억이 쉽고, 10억보다 100억이 쉽다 (커리어도 마찬가지).

### 연봉 결정 공식
> "내 연봉은 협상 테이블에서 정해지지 않아. 그 회사의 연매출에 의해서 결정돼. 공식: 연매출 / 직원 수 = 1인당 매출."
- 네카라쿠배 연봉이 높은 이유 = 1인당 매출이 높음.
- 연봉 올리려면: 1인당 매출 높은 회사로 이직 OR 내 단계 올리기.

### 노력 vs 성과
- 노력 = 시간, 애를 씀. 한자: 努(힘쓸), 力(힘).
- 성과 = 결과를 맺음. 매출↑, 재구매율↑, 시장 반응 변화, 효율↑.
- **"노력한 사람에겐 밥과 술. 성과 낸 사람에겐 돈."**
- 트루스의 성과 정의: "이전보다 *훨씬* 나아진 결과" = 동종업계 평균 초월.

### 참여 vs 주도
- **주도성 두 가지**:
  (1) 기준을 세울 수 있는가
  (2) 책임질 수 있는가 = 개선할 수 있는가
- 회사엔 매출·재방문율·원가율·별점 등 지표 多. 그중 무엇이 최대 이익 기여인지 *선택*하는 게 주도성.

## 5. 바운더리 — 트루스 일하기의 핵심 도구

### 우선순위 vs 바운더리
- **우선순위** = 순서대로. 결국 2순위 이후도 함.
- **바운더리** = 핵심 하나만. 나머지는 *안* 함.

### 바운더리 3단계
1. **정한다** (시간/기간을 정함 — 쉬움)
2. **가지치기 한다** (제일 어려움. 뭘 쳐낼지)
3. **반복한다** (가지치기 잘되면 자동)

### 단계별 바운더리 팁
| 커리어 단계 | 바운더리 핵심 |
|---|---|
| **시간 단계** | 시그널과 노이즈 구분. "암기 안 되는 건 내 것이 아니다." |
| **성과 단계** | 2:8 법칙. 번아웃 = 뭐가 중요한지 모르는 상태. |
| **자본 단계** | 1원칙 사고 (스페이스X 로켓: "되돌아오게 한다"). 음식점 1원칙: 싸고 맛있게. 러쉬: "매출 최대, 비용 최소." |

### 사례
- **스웨덴 피크닉 와인바**: 1원칙 = "와인 제일 싸게 판다" → 내추럴 와인 독점.
- **러쉬코리아 2025**: 매출↑ 영업이익↓ → 1원칙 "매출 최대, 비용 최소" → 6개월 만에 영업이익 목표 달성.
- **러쉬 광고 행사**: 2024년 연 23개 → 2025년 4개로 가지치기 → 페스티벌·팝업 모두 바이럴 + 예산 남음.

### 트루스 부부 워크샵 4가지 (반복)
1. **회고**
2. **숫자언어** (노동소득·자본소득 목표)
3. **바운더리** (3개월로 성과 낼 수 있는 일)
4. **고객언어** (고객에게 듣고 싶은 말)

### 인큐의 실패 → 브레이닝의 성공
- 인큐: 마구잡이 전문가. 책 읽으면 수업이 만들어짐. 수업 종류 어마어마. 2달마다 220명 모객. 주 7일 80시간.
- "참 좋은데 설명할 방법이 없네." → 시장 안에서 포지셔닝 X.
- 인큐 버리고 → 아이들 논술 (브레이닝) → 한 번 모으면 1~5년. 대기자 100명.
- 인큐 시절 일론 머스크도 주 7일 80시간. 우리도 주 7일 80시간. 그는 전 세계 1위 / 우리는 길을 잃음. **차이 = 바운더리.**

### 5분 룰 + 한 가지 집중 훈련
- "포스트잇에 바운더리 단어 하나만 적어. 끝점을 숫자언어로."
- 3분 → 5분 → 10분 침묵 검토.
- "5분째가 짧게 느껴지면 뇌가 가속화되기 시작."

## 6. 키신 (Key Scene) — 기획 꿀팁

### 키신을 만들어라
- 책 *<SAVE THE CAT>* 영감.
- 주인공 첫 등장 시 고양이 구하는 장면처럼, *한 장면*으로 모든 걸 전달.
- 끝에서 일한다 → 그 끝의 *한 장면*을 그리는 것이 키신.

### 키신은 글이 아니라 그림으로
- 글 = 어른의 뇌. 그림 = 아이의 뇌.
- 머릿속에 각인되어야 상대에게 전달 가능.
- **"좋은 리더는 자기가 기억하는 장면을 그림 그리듯 생동감 있게 설명한다."**

### 사례
- 러쉬 페스티벌 화장실: 키신 = 대표가 *맨손으로 청소*하는 장면 (고무장갑 절대 X).
- 페스티벌 마지막: 키라라 시그널송에 떼창 → 그 순간 바이럴 터짐.
- 흑백요리사: 안대 쓰고 블라인드 테스트, 두부가 하늘에서 내려오는 장면.

### 평소 훈련
- 여행·프로젝트 시작 시 키신부터.
- 머리에 집착하는 장면 하나. "이것만 되면 나머진 다 괜찮아."

## 7. 이등병 헨리 (10년 실천 비결)

단계별로 헨리(자기 일을 끝까지 책임지는 사람)를 다르게 운영:
- **시간 구간**: 내가 헨리가 됨 (내 에너지로 돈 벌기)
- **성과 구간**: 팀에 헨리를 심음 (헨리가 성과 내게)
- **자본 구간**: 헨리가 될 수 있는 *환경 자체*를 팖

> 데스커 라운지 프로젝트: 팀이 들어가기 전 헨리 교육 2달. "나를 판 게 아니라 헨리로 돌아가는 팀을 판 거야."

## 11. 고객 언어 (Appendix 본문)

### 고객 언어는 반드시 *구어체*
- 사피엔스(유발 하라리): 뇌는 *구어*를 기억함.
- 문어체: "화장실 청결을 유지해야 합니다."
- 구어체: "개 힘들었다", "빨리 매점 가자."

### 고객 언어 100개 이상 모으는 게 경쟁력
- 카페: "일하기는 여기가 최고야. 히크앳눙크 없으면 어쩔 뻔."
- 피부과: "진짜 필러 맞을 걸 그랬어요."
- 헬스장: "24시간 해서 좋아."
- 기획자는 이걸 갖고 있어야 함.

### 듣고 싶은 말을 *먼저* 정한다
> "내 시선이 나한테 안 가. 모든 시선이 고객의 입에만 가 있게 돼. 가지치기도 거기서 나와."

### 사례
- 러쉬 페스티벌 화장실: 잡았던 고객언어 "러쉬가 킥이네" → 실제로 들은 말 "러쉬가 헤드라이너네"
- 추가: "내 방보다 깨끗해", "옆집 아줌마도 찾아봤대요"
- 생각 마라톤 자체의 고객언어: "벌써 대기자가 천 명이래"
- 소울정 유튜브: "이걸 공짜로 봐도 되나요?"
- 브레이닝: "우리 애가 나보다 글을 잘 써"

## 13. 위키 페이지에 들어갈 핵심 다이제스트 (5줄 요약)

1. **트루스 = "한 번 더, 함께, 끝까지 생각한다"는 메타 사고를 가진 부부 공동체의 working method.** 끝까지 생각 = 숫자 언어 + 고객 언어 (구어체). 둘 다 나오지 않으면 시작하지 않는 게 원칙.
2. **돈은 무의식·감정 → 숫자 언어로.** 돈은 소유가 아니라 *학습의 도구*. AGI 시대엔 노동소득 점프 + 자본소득 업그레이드 둘 다 필수. 자산 3공식 = 절제 → 시간 → 투자.
3. **노동소득 커리어 6단계 (시간/성과/자본)는 트루스의 자기 진단 도구.** 각 단계에서 *무엇을 모르는지*가 다르다. 위로 갈수록 공급은 1/10 토막. 연봉은 협상이 아니라 1인당 매출이 결정.
4. **바운더리 = 일하기의 핵심 도구.** 우선순위 ≠ 바운더리. 바운더리는 핵심 하나만, 나머지 *안* 함. 3단계: 정한다 → 가지치기 → 반복. 단계별 도구: 시그널/노이즈 → 2:8 법칙 → 1원칙 사고.
5. **학습 환경은 설계 가능.** 3메들리 1터치, 4역할(응원/기록/행동/질문), 명함·단톡방 금지. 무의식은 환경·감정·반복으로 바꿈. 학습 5단계 = 지행용훈평.
