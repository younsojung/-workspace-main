## 2. 첫 화면 ~ CTA 동선 분석 (Above-the-fold 흐름)

페이지 총 높이 **7,021px**. 데스크탑 기준 8개 섹션으로 나눠진다.

### Section 1 (0~900px) — 히어로
```
[배경] 어두운 톤, 책장 흩날리는 듯한 이미지
[메인 카피, 분홍 손글씨]  "나의 곁에서는 땀냄새가 납니다."
[서브 카피]  SNS에 공개하기는 어려웠습니다.
            하지만 날것의 15년동안 성장과정을 기록했습니다.
[CTA] 없음 (시각적 큐브 1개)
```
**문제점**:
- 시각적으로 무겁고, **"이게 뭐 하는 사이트인지"** 1초 안에 안 들어옴
- "생각구독"이라는 상품명이 히어로에 노출되지 않음
- 첫 화면에서 CTA가 없어 **이탈률이 높을 가능성**

### Section 2 (900~1800px) — 3컬럼 가치제안
```
[좌] "내가 경험한 것만 / 내가 실천한 것만"
     15년째 생각을 글로 쓰는데 여전히 두렵습니다.
     생각만 하는 사람이 되지 않을까?
     그래서 매일밤 실천한 만큼만 씁니다.

[중] "날것 그대로 솔직하게"
     자리에 앉아서 꺼이꺼이 울면서, 나의 생각을 현실로 만들
     준비를 하는거야. 나의 부끄러움을 토해내고, 날것의 나를
     그대로 인정하고, 용기있게 또 일을 벌리는 거지.

[우] "한장, 두장, 세상 / 그렇게 성장"
     그렇게 각자의 생각대로, 삶의 결대로
     각자의 한장, 두장, 세장 그리고 성장을
     쌓는 중이라는 이야기를 남겨두고 싶었다.
```
**카피 분석**:
- 3컬럼 모두 **에세이/철학** 톤. **상품 가치(왜 사는지)** 가 아니라 **작가 신념**을 말함
- 잠재 고객 기준 의문: "그래서 내가 뭘 얻는가?"가 답되지 않음
- 톤이 일관되게 **3인칭/관조적**이라 윤소정의 1인칭("나의 영혼의 친구") 보이스와 어긋남

### Section 3 (1800~2700px) — 브릿지 카피
```
[메인 카피, 손글씨]  "나의 곁에서는 땀냄새가 납니다."  (히어로와 중복!)
[서브] 날것의 성장과정을 sns에 공개하기는 어려웠습니다.
       15년째 생각을 글로 쓰지만 여전히 두렵습니다.
       생각만 하는 사람이 되지 않을까? 그래서 매일밤 실천한 만큼만 씁니다.
[CTA 1] "그녀의 이야기가 궁금하다면?"  →  추정 /About or /firstthought
```
**문제점**:
- **히어로와 똑같은 카피가 반복**됨 ("땀냄새가 납니다") — 스크롤 보상이 없음
- "그녀" = 윤소정 본인을 3인칭화 → 사이트 운영자가 누구인지 모호하게 느껴짐
- CTA가 모호("이야기가 궁금하다면?")해서 다음 단계 기대치를 못 만듦

### Section 4 (2700~3600px) — 핸드라이팅 사진 + 손글씨 질문
```
[좌측 세로 손글씨]  "나의 곁에서는" (반복)
[중앙 사진]  손글씨로 노트에 글 쓰는 손
[하단 손글씨]  "그녀의 생각이 궁금하다면?"
```
**문제점**:
- **또 다시 "그녀의 ~"** 동일 패턴의 카피
- CTA 텍스트가 약함 (호기심 트리거만, 결제/가입 행동 유발 X)

### Section 5 (3600~4500px) — 2개의 핵심 CTA
```
[CTA Block 1]
  생각을 현실로 만드는 그녀가
  14년간 한 장, 매월 한 권씩 기록한 날것의 글
  → [버튼] "과거의 기록, 생각구독 ebook 보러가기"
  (→ /esingle?utm_source=imweb&utm_medium=banner&utm_campaign=homeebook)

[CTA Block 2]
  이번달 업로드 되는, 그녀의 가장 최근의 생각은?
  (생각구독은 매월 말일 24시간만 OPEN)
  → [버튼] "생각구독 오픈 알림 신청하러 가기"
  (→ /subscribeopen?utm_source=imweb&utm_medium=banner&utm_campaign=homesubopen)
```
**여기가 사실상 메인 페이지의 진짜 CTA 구역**.

**문제점**:
- **CTA가 페이지 절반쯤(3,600px)에서야 처음 등장** — 너무 늦음
- **2개 CTA의 우선순위 불명확** — "ebook 보러가기"와 "구독 알림" 중 무엇이 메인?
- "매월 말일 24시간만 OPEN"은 **희소성 후크가 강한데** 너무 작게/늦게 노출됨

### Section 6 (4500~5400px) — 클로징 비주얼
```
[배경] 펜과 종이의 흐릿한 이미지
[손글씨]  "우리의 생각이 될 때까지"
[서브]  생각구독과 함께한 친구들의 한줄평
        나의 생각이 우리의 생각이 될 때까지
```
**문제점**:
- "한줄평"이라고 했는데 실제 한줄평은 다음 섹션의 별도 리뷰리스트 → **레이아웃 끊김**
- 다시 한 번 손글씨/감성톤 — 페이지 전체가 **감정 톤만 누적**되고 **구체적 가치**는 못 쌓음

### Section 7~8 (5400~7021px) — 리뷰 리스트
```
- [ebook 제목] + 짧은 한줄평 + 닉네임 + 날짜 + 5점 별
- 약 20개 노출 (대부분 5점)
- 모두 javascript:NEWEST_POST.viewReviewPostDetail(...) 링크
```
**문제점**:
- **리뷰가 페이지 끝 절반(약 2,400px)을 차지** — 신뢰지표인데 너무 길어 결정 피로
- **리뷰 → 결제 동선 없음**: 리뷰 클릭하면 모달만 뜸, 구매로 이어지지 않음
- 책 표지는 보이지만 **별점/판매가격/구매버튼이 같은 자리에 없음**

## 3. 카피·메시지 현황 — 톤·핵심 단어 맵

### 반복되는 키 메시지 (페이지 전체)
- "나의 곁에서는 땀냄새가 납니다" — **3회 반복**
- "그녀" — 윤소정 본인을 지칭, **4회 이상 반복**
- "날것" — 핵심 브랜드 단어 ✅ (e.g. 책 제목 "[E-book] 날것의 사랑")
- "생각" — 모든 상품/콘텐츠의 공통 단어 ✅
- "매일밤 실천한 만큼만 씁니다" — 작가 신뢰 빌더 ✅
- "매월 말일 24시간만 OPEN" — 희소성/긴급성 후크 ✅ (그러나 약하게 노출)

### 톤
- **3인칭 관조형** ("그녀의 생각이 궁금하다면?")
- **손글씨 + 한자·고어체 분위기** — 감성/문학적
- **저녁시간/명상/작업실** 무드 (다크 배경, 회색조)
- **카피=시(詩)**: 정보 전달이 아니라 분위기 전달 중심

### 윤소정봇 1인칭 톤(<생각구독> 본문)과의 갭
- 봇은 "안녕? 나의 영혼의 친구" — **친밀한 1인칭**
- 사이트는 "그녀" — **거리감 있는 3인칭**
- → 본문을 읽으면 친밀해지지만, **랜딩에서는 거리가 느껴짐**

### 누락된 메시지 (없는 것)
- **"구독료 / 발행 주기 / 무엇이 들어있는지"** — 가격·스펙 정보 0
- **"이런 분께 추천"** — 페르소나 매칭 없음
- **"몇 명이 구독중인지"** — 사회적 증거 숫자 없음 (4K 구독자, 유튜브 20만 등 외부 자산이 사이트에 안 보임)
- **"첫 호 미리보기 / 샘플"** — 트라이얼 진입점 없음
- **"FAQ / 환불정책"** — 결제 직전 안심 신호 없음

## 4. 약점 진단 — 우선순위 TOP 5

### A. 첫 화면이 "무엇을 파는 곳인지" 안 알려줌 (최우선)
- 히어로에 **상품명(생각구독), 가격, 가치제안 한 줄** 부재
- "땀냄새가 납니다"는 강렬하지만 **상품 인지를 0초 안에 못 만듦**
- → 신규 방문자(유튜브 20만 ↔ 인스타 8만에서 유입) 이탈률 높을 가능성

### B. CTA가 너무 늦고, 우선순위가 둘로 쪼개짐
- 페이지 절반(3,600px)에서야 첫 CTA 등장
- "ebook 보러가기" vs "구독 알림 신청" 동등 가중치 → 사용자가 결정 못 함
- 메인 상품(생각구독)이 메뉴 7번째에 묻혀있음

### C. 메뉴/페이지 구조 중복
- 처음이라면 / firstthought (2개)
- ebook / etimeline (2개)
- btopic / btimeline (2개)
- `/95`, `/102030` 같은 비의미적 URL
- → 사용자 동선이 분기점마다 헷갈림

### D. 톤 불일치 — 사이트는 3인칭 시(詩), 본문은 1인칭 편지
- "그녀의 생각이 궁금하다면?" 같은 카피는 **신규 방문자에게는 작가가 누구인지 모호**
- 사이트가 "윤소정 박물관" 같은 톤인데, 실제 콘텐츠는 "친구가 보내는 편지"
- → **기대치와 경험 사이 불일치** 발생

### E. 가격/스펙/사회적 증거 미노출
- 구독료 정보 없음 (월 얼마? 1회 결제? 자동결제?)
- 4K 구독자, 유튜브 20만, 인스타 8만 같은 **외부 자산이 페이지에 안 보임**
- "이런 분께 추천" / FAQ 없음 → **결제 직전 마찰점**

### F. 부차적 — 리뷰 리스트 비대
- 페이지 하단 6000px 중 2400px이 리뷰
- 리뷰 클릭 → 모달만 → 구매전환 동선 없음
- 리뷰는 신뢰지표지만 **상품 단위(책 1권)**가 아니라 **메인 페이지에 쌓이는** 구조라 분산

## 5. 개편 0단계 — 우선순위 권장 (실행 후보)

> 0단계 진단이므로 결정 X, 옵션만 제시.

### Track 1: 히어로 한 줄 다시 쓰기
- 현재: "나의 곁에서는 땀냄새가 납니다."
- 후보: "매월 1권, 19년차 작가의 날것 생각. 구독자 4,000명." (스펙 + 사회증거)
- 또는: "안녕? 나의 영혼의 친구. 매달 한 권씩 보내는 편지." (봇 톤과 일치시키기)

### Track 2: 메인 CTA 단일화
- "구독 신청"을 1순위 CTA로 통일 (메뉴/히어로/하단 모두)
- "ebook 보러가기"는 보조 CTA (sticky bar나 secondary)

### Track 3: 메뉴 정리 (13개 → 5~6개)
- 처음이라면 (About + firstthought 통합)
- 생각구독 (subscribe + subscribeopen 통합) **← 가장 큰 버튼**
- 책장 (ebook + etimeline + book + 80편 통합)
- 글 아카이브 (btopic + btimeline + 102030 통합)
- 오프라인 (insideroom + 95 통합)
- 커뮤니티

### Track 4: 사회적 증거 모듈 신설
- 히어로 직후 한 줄: "현재 4,127명이 구독 중 · 매월 말일 24시간만 OPEN"
- 유튜브 20만 / 인스타 8만 / 누적 80권 ebook 같은 숫자 노출

### Track 5: 리뷰 영역 압축

## 8. 비교 참조용 — 윤소정봇 톤 vs 사이트 톤

| 측면 | 윤소정봇 (생각구독 본문) | 사이트 (랜딩) |
|---|---|---|
| 인칭 | "안녕? 나의 영혼의 친구" (1인칭, 너) | "그녀의 생각이 궁금하다면?" (3인칭) |
| 거리 | 친밀, 편지 | 관조, 시(詩) |
| 정보 | 풍부, 직설 | 분위기, 암시 |
| 행동유도 | 강함 (보내는 글) | 약함 (호기심) |

→ **개편 방향 후보**: 사이트 톤을 봇 톤에 맞춰 1인칭화하거나, 봇 톤은 그대로 두고 사이트 톤을 "신규 진입자용 안내" 톤으로 정리
