# 40-personal

> **개인 기록.** 날짜 기반, 회고, 아이디어, 할 일.

## 하위 폴더

| 번호 | 폴더 | 용도 | 파일명 형식 |
|------|------|------|-------------|
| 41 | daily | Daily Notes | `YYYY-MM/YYYY-MM-DD.md` |
| 42 | weekly | Weekly Reviews | `YYYY-WXX.md` |
| 43 | ideas | 아이디어 캡처 | 자유 |
| 44 | reflections | 회고/학습 | 자유 |
| 46 | todos | 할 일 관리 | `active-todos.md` |

## Daily Note 리듬

- 매일 아침 또는 저녁: `daily-note` 스킬로 자동 생성
- 월별 폴더 구조: `41-daily/2026-04/2026-04-24.md`
- 템플릿: `00-system/01-templates/daily-note-template.md`

## Weekly Review

- 주 1회 (일요일 또는 월요일)
- `weekly-synthesis` 스킬로 자동 생성
- 주간 배운 것, 다음 주 방향 정리

## Todos

- `46-todos/active-todos.md`에 현재 활성 할 일만
- 완료 항목은 체크마크 후 일정 주기로 정리
- `todo`, `todos` 스킬로 추가/조회
