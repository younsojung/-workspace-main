# 00-inbox

> 임시 캡처 공간. **영구 저장소 아님.**

## 원칙

- 생각나는 즉시 기록 (아이디어, 링크, 빠른 메모)
- **20개 미만 유지**
- **주 1회 정리** — 적절한 폴더로 이동 또는 삭제

## 처리 흐름

1. **Capture**: 아무 형식이나 일단 여기 던지기
2. **Process**: 주간 단위로 Claude에게 "inbox 정리해줘" 요청
3. **Organize**: 각 항목을 위치로 이동
   - 프로젝트 아이디어 → `10-projects/`
   - 지식/학습 → `30-knowledge/` 또는 `30-knowledge/00-wiki/`
   - 개인 메모 → `40-personal/`
   - 참고 자료 → `50-resources/`

## 팁

- 파일명은 대충 써도 됨 (어차피 이동할 거)
- `inbox-YYYY-MM-DD.md` 같이 날짜 prefix로 쌓아도 됨
