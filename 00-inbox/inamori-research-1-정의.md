# 이나모리 가즈오의 채산표(採算表) — 리서치 결과

> 조사 범위: 한국어 + 영어 + 일본어 병행 검색 / 독립 소스 5개 이상 교차검증
> 작성일: 2026-05-19

---

## 요약 (3줄)

1. 채산표는 아메바(소집단) 단위의 월간 독립채산 보고서로, 핵심 지표는 "시간당 부가가치 = (매출 - 인건비 제외 경비) ÷ 총노동시간"이다. 일반 손익계산서(P&L)와 달리 인건비를 비용으로 보지 않고 부가가치 창출자로 취급한다.
2. 1963년 교세라 시가 공장 개설을 계기로 탄생. 서로 성격이 다른 두 제조 부문을 공정하게 비교할 단일 지표가 필요해 "시간당 부가가치"를 고안했고, 이것이 아메바경영의 회계 엔진이 됐다.
3. 한국에서는 메리츠금융그룹 등이 도입 사례로 언급되며, "이나모리 가즈오의 회계 경영" 등 번역서 4종 이상이 출간돼 있다.

## 1. 채산표의 정의

**팩트**:
채산표는 교세라가 아메바(5~50명 소집단) 단위로 매월 작성하는 관리회계 보고서다. 이나모리는 "전사원이 쉽게 이해할 수 있는 가계부와 같은 채산표"라고 정의했다. 정식 명칭은 "시간당 채산표(時間当たり採算表, Hourly Efficiency Report)".

**구조 — 생산 아메바 기준 항목**:

| 항목 | 내용 |
|------|------|
| 총생산고(Gross Production) | 사내 타 아메바 판매(Internal Sales) + 사외 생산(Outside Production) 합산 |
| 사내 구매(Internal Purchases) | 타 아메바로부터 구매한 부품·서비스 비용 |
| 순생산고(Net Production) | 총생산고 - 사내 구매 |
| 경비 차감(Deductions) | 인건비를 제외한 모든 비용(재료비, 외주비, 감가상각 등) |
| 부가가치(Added Value / 差引利益) | 순생산고 - 경비 차감 |
| 총노동시간(Labor Hours) | 정규시간 + 잔업시간 합계 |
| **시간당 채산(Hourly Efficiency)** | **부가가치 ÷ 총노동시간** |

**일반 P&L과의 결정적 차이 3가지**:
- **인건비 처리**: 일반 P&L은 인건비를 비용으로 차감. 채산표는 인건비를 경비에서 제외하고, 사람을 부가가치 창출의 주체로 본다.
- **측정 단위**: 일반 P&L은 회사 전체·사업부 단위, 월/분기/연 기준. 채산표는 5~50명의 아메바 단위, 매월 예측 vs. 실적 비교.
- **지표**: P&L의 최종 지표는 순이익(Net Income). 채산표는 "시간 효율(시간당 부가가치)"이 최종 지표다.

## 2. 탄생 배경 — 1963년, 시가 공장 문제

**팩트**:
1963년 교세라는 본사 교토 공장과 별도로 시가(滋賀) 공장을 신설했다. 두 공장은 생산 방식이 달라 단순 이익 비교가 불가능했다. "어느 쪽이 더 잘하고 있는가"를 공정하게 측정할 단일 지표가 필요했고, 이나모리는 "부가가치"라는 개념을 도입했다. 인건비를 제외한 경비를 차감한 뒤 총노동시간으로 나눠 "단위 시간당 얼마의 부가가치를 만들었는가"를 비교한 것이 시간당 채산표의 시작이다.

**아메바경영과의 관계**:
채산표는 아메바경영의 "신경계"다. 아메바경영은 조직을 5~50명의 소집단으로 나눠 각 집단이 독립채산 미니 기업처럼 운영되도록 하는 경영 시스템이다. 채산표 없이는 아메바경영이 작동하지 않는다. KCCS(교세라 커뮤니케이션 시스템)는 이 시스템을 전 세계 400개 이상 기업에 컨설팅 도입했다.

**출처**: [이나모리 공식 사이트 — 시간당 채산제도 탄생 (1963년)](https://www.kyocera.co.jp/inamori/archive/episode/episode-15.html) / [KCCS 아메바경영 소개](https://biz.kccs.co.jp/service/amoeba-about/)
**확인 수준**: 확인됨 (1차 소스, 공식 사이트)

## 3. 시간당 부가가치(時間当たり付加価値) — 계산법

**팩트**:

```
시간당 부가가치(채산) = { 생산고(매출) - 제경비(인건비 제외) } ÷ 총노동시간(정시간 + 잔업)
```

- **생산고**: 아메바가 시장(외부) 또는 사내 타 부문에 판매한 총액
- **제경비**: 재료비, 외주비, 감가상각비, 여비교통비 등. 인건비는 포함하지 않음
- **총노동시간**: 소속 전원의 정규 + 잔업 시간 합산

**의미**: 수치가 높을수록 같은 시간 투입으로 더 많은 부가가치를 창출한다는 의미. 아메바별 목표 수치를 사전에 설정하고, 월말 실적과 비교해 전원이 확인한다.

**부연**: 이나모리는 "연필 한 자루, 클립 하나도 시간당 채산에 영향을 미친다"며, 이 지표를 모든 직원이 체감적으로 이해해야 한다고 강조했다.

## 4. 이나모리 회계학 7원칙

이나모리 공식 한국 사이트 및 교보문고·알라딘 등에 번역 출판된 "이나모리 가즈오의 회계 경영"(2022년 국내 출간)에서 동일하게 확인됨.

| # | 원칙명 | 핵심 내용 |
|---|--------|-----------|
| 1 | 캐시 베이스 경영의 원칙 | 현금흐름 중심 판단. 복잡한 회계 기법보다 실제 현금 움직임 우선 |
| 2 | 일대일 대응의 원칙 | 물건·돈 이동 시 전표가 반드시 일대일 대응. 허위·지연 기록 불허 |
| 3 | 근육질 경영의 원칙 | 불필요한 자산 없는 건전한 재무 상태 유지 |
| 4 | 완벽주의의 원칙 | 회계 숫자는 100% 정확해야 함. 타협 없음 |
| 5 | 더블 체크의 원칙 | 복수 인원의 상호 확인으로 실수·부정 방지 |
| 6 | 채산향상의 원칙 | 아메바경영으로 전 직원이 채산 향상에 참여. 채산표가 핵심 도구 |
| 7 | 투명 경영의 원칙 | 회사 실태를 전 직원에게 공개. 채산표도 전원에게 공개 |

**상위 원칙**: 7원칙의 전제는 "매상 최대, 경비 최소"(売上最大、経費最小). 이익은 두 가지의 차이에서 나오는 결과이지, 목적 자체가 아니라는 철학.

**출처**: [이나모리 회계학 7원칙 — 공식 한국 사이트](https://korea.kyocera.com/inamori/management/accounting/accounting01.html) / [매출 최대 경비 최소 — 이나모리 철학](https://korea.kyocera.com/inamori/philosophy/words64.html)

## 한계 / 미확인 사항

1. **채산표 실제 양식**: 공개 소스에서 실제 표의 전체 항목과 행 구성을 담은 이미지나 서식을 찾지 못했다. O'Reilly 원서 Chapter 4에 있는 것으로 확인되나 유료 구독 필요.
2. **한국 도입 기업 상세**: 메리츠금융그룹 외 구체 기업명·도입 성과는 공개 자료 없음. 단일 소스 언급에 그침.
3. **"채산성 향상의 원칙"이 7원칙 중 6번인 점**: 소스별로 번호 순서나 명칭 표현이 미세하게 다름. 공식 한국 사이트 기준으로 정리.
4. **시간당 부가가치 목표값**: 교세라 내부 기준치(예: 시간당 몇 엔/원)는 공개 정보 없음.
