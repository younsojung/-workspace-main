# 이나모리 가즈오식 개인 채산표: 리서치 결과

> 리서치 일시: 2026-05-19
> 검색 언어: 한국어 + 영어 + 일본어
> 검증 소스: 6개 독립 출처

---

## 요약 (3줄)

- 이나모리의 "시간당 채산" 공식은 기업 단위로 설계됐지만, 개인에게 직접 이식 가능한 논리 구조를 갖는다. "시간당 부가가치 = (수입 - 비용) ÷ 투입 시간"이 핵심이다.
- 개인 채산표는 단순 가계부(수입-지출)와 근본적으로 다르다. 시간이라는 변수가 들어가면서 "얼마나 벌었나"가 아닌 "1시간으로 얼마만큼의 가치를 만들었나"라는 질문으로 바뀐다.
- "개인 채산표"라는 명칭의 공식 사례나 표준 양식은 현재까지 확인되지 않음 (한국어·영어·일본어 검색 모두 동일). 이나모리 원리에서 개인용으로 합리적으로 연역한 응용 양식을 이 문서 하단에 제안한다.

## 1. 개인 채산표의 개념이 존재하는가?

**팩트**: "個人採算表(개인 채산표)"라는 용어로 검색했을 때 직접 사례는 발견되지 않았다. 이나모리 공식 사이트, 일본 아메바 경영 관련 서적, 한국어 블로그 어디에서도 이 명칭이 쓰인 적이 없다. 단, 이나모리 스스로가 채산표를 "가계부처럼 이해하기 쉽게" 설계했다고 언급하며, 개인 직원 한 명 한 명의 일상이 채산표에 축적된다고 말한 기록은 공식 사이트에서 확인된다.

**출처**: [日々採算をつくる, 稲盛和夫 Official Site](https://www.kyocera.co.jp/inamori/philosophy/words65.html)

> "채산표는 한 사람 한 사람의 매일의 삶의 방식이 누적된 결과다."

**확인 수준**: 확인됨 (직접 인용). "개인 채산표" 응용 사례는 불확실.

## 2. 아메바 채산표의 원래 구조

**팩트**: 이나모리가 1963년 교세라에 도입한 "시간당 채산 제도(時間当り採算制度)"의 핵심 공식은 다음과 같다.

```
시간당 채산 = (총생산 - 경비) ÷ 총 근무시간
            = 부가가치 ÷ 총 근무시간
```

채산표의 구성 항목 (기업용 원형):

| 구분 | 항목 |
|------|------|
| 총생산(수입측) | 매출액, 내부 판매액(아메바간 거래) |
| 경비(지출측) | 원재료비, 외주비, 감가상각비, 제조경비 (단, 인건비 제외) |
| 부가가치 | 총생산 - 경비 |
| 시간당 채산 | 부가가치 ÷ 총 근무시간 |

인건비를 경비에서 제외하는 이유: "사람은 비용이 아니라 가치 창출의 원천"이라는 이나모리의 철학 때문. 따라서 시간당 채산을 올리는 3가지 방법은 ①총생산 증가 ②경비 억제 ③효율 향상으로 노동시간 단축이다.

**출처**: 
- [시간당 채산 제도의 도입, 이나모리 공식 한국 사이트](https://korea.kyocera.com/inamori/profile/episode/episode04.html)
- [Amoeba Management Wikipedia](https://en.wikipedia.org/wiki/Amoeba_Management): "Profit per hour = (sales - cost) ÷ working hours"
- [Springer Nature: Amoeba Management and Hourly Profit System](https://link.springer.com/content/pdf/10.1007/978-981-19-3398-1_6)

**확인 수준**: 확인됨. 공식 및 항목 구조 다수 소스 교차 검증 완료.

## 3. 가계부와의 본질적 차이

**팩트**: 

| 구분 | 일반 가계부 | 채산표 방식 |
|------|------------|------------|
| 핵심 질문 | "얼마 벌고 얼마 썼나?" | "1시간으로 얼마 만들었나?" |
| 시간 변수 | 없음 | 있음 (분모) |
| 평가 단위 | 금액 (월/연) | 시간당 부가가치 |
| 목적 | 지출 통제 | 가치 창출 극대화 |
| 노동 취급 | 수입의 결과 | 시간을 자원으로 봄 |

이 차이가 중요한 이유: 가계부는 "번 돈을 어떻게 쓰냐"를 관리하지만, 채산표는 "내 시간 한 단위가 얼마의 가치를 만들었나"를 관리한다. 같은 월 500만원 수입이라도 100시간 투입과 200시간 투입은 완전히 다른 결과다.

- [클로브 블로그, 경영자의 시간 가치와 기회비용](https://clobe.ai/blog/calculating-manager-hourly-rate-opportunity-cost)

## 5. 인생 방정식과 채산표의 연결

**팩트**: 이나모리의 인생 방정식은 다음과 같다.

```
인생・일의 결과 = 사고방식(-100 ~ +100) × 열의(0 ~ 100) × 능력(0 ~ 100)
```

채산표와의 연결 지점:
- **능력**: 시간당 기술 단가에 반영 (고숙련 = 높은 시간당 가치)
- **열의**: 투입 시간과 질에 반영 (높은 몰입 = 같은 시간에 더 높은 산출)
- **사고방식**: 어떤 활동에 시간을 배분하는가를 결정 (전략적 집중 vs. 소모)

이나모리는 방정식의 요소 중 사고방식이 유일하게 마이너스 값이 된다고 강조한다. 채산표로 보면, 아무리 능력(시간당 단가)이 높아도 잘못된 사고방식으로 수익성 낮은 활동에 시간을 쏟으면 전체 부가가치는 역전된다.

- [Corporate Rebels: Inamori's Equation](https://www.corporate-rebels.com/blog/success-in-work-and-life-attitude-x-effort-x-ability)

**확인 수준**: 확인됨 (방정식). 채산표와의 연결은 합리적 연역 (불확실 수준 아님, 논리적으로 정합).

## 6. 직장인/프리랜서/사업가별 시간당 부가가치 계산

**직장인**:
```
시간당 부가가치 = (월 급여 - 업무 관련 직접 지출) ÷ 실제 투입 시간
```
주의: 회사가 정한 근무시간이 아닌, 출퇴근/야근 포함 실제 시간으로 계산. 월 250만원 급여 + 교통비 30만원 지출, 주 50시간 × 4주 = 200시간이면 시간당 가치 = 110,000원.

**프리랜서**:
```
시간당 부가가치 = (수주 금액 - 직접 경비) ÷ 해당 프로젝트 투입 시간
```
한국 기준 경력별 목표 단가: 초급(1-3년) 3-5만원, 중급(3-7년) 5-8만원, 시니어(7-10년) 8-12만원/시간.
[source: 프리랜서 단가 산정 가이드, jeff-info.co.kr]

**사업가**:
```
목표 시간당 가치 = 연 목표 매출 ÷ 연간 가용 업무 시간
```
예: 연 5억원 목표, 연 2,000시간 투입 → 목표 시간당 가치 = 250,000원.
이 기준치보다 낮은 가치의 업무는 위임 또는 외주 판단 기준이 된다.
[source: 클로브 블로그, 경영자의 시간 가치](https://clobe.ai/blog/calculating-manager-hourly-rate-opportunity-cost)

**확인 수준**: 각 계산식 확인됨. 금액 예시는 출처 있음.

## 7. 개인이 채산표를 써야 하는 이유

이나모리의 말을 그대로 가져오면:

> "매일의 숫자를 보지 않고 경영하는 것은 계기판을 보지 않고 비행기를 조종하는 것과 같다."
> — [日々採算をつくる, 稲盛和夫 Official](https://www.kyocera.co.jp/inamori/philosophy/words65.html)

이것이 개인에게도 그대로 적용된다. 내 삶을 경영한다면, 내 시간 한 단위당 얼마의 가치를 만들었는지를 보는 것이 계기판이다.

개인 채산표를 써야 하는 구체적 이유:
1. **시간 배분 결정**: 어떤 활동이 진짜 가치를 만드는지 숫자로 확인
2. **사이드 프로젝트 수익성 판단**: 감각이 아닌 시간당 부가가치로 판단
3. **아웃소싱 기준 설정**: 내 시간당 가치보다 싼 비용이면 위임
4. **자기 착취 방지**: 바쁘지만 가치가 낮은 활동을 발견
5. **성장 추적**: 같은 시간으로 더 높은 가치를 만들어가는 궤도 확인

## 한계 / 미확인 사항

- "個人採算表" "개인 채산표"라는 명칭 자체를 쓴 공식 사례, 책, 강의는 한국어·영어·일본어 모두에서 발견되지 않음. 이 리서치에서 제안하는 양식은 이나모리 원리의 연역 응용임.
- 이나모리의 채산표에서 정확한 경비 세부 항목 목록은 교세라 공식 사이트에 공개되지 않음. 서적(『아메바 경영』) 원문 확인 필요.
- 개인이 실제로 채산표를 작성해본 후기 블로그나 커뮤니티 글은 일본어/한국어 모두 발견되지 않음. 직접 실험 후 검증 필요.
