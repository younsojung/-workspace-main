# 이나모리 가즈오식 채산표(採算表) — 회사 단위 작성법 리서치

> 리서치 완료일: 2026-05-19
> 검색 언어: 한국어 + 영어 + 일본어
> 교차검증 소스 수: 8개 이상

---

## 요약 (3줄)

- **채산표의 핵심 구조**: 총출하(사외+사내) - 사내구매 - 경비 = 부가가치(차인매출) → 부가가치 ÷ 총시간 = 시간당 채산. 노무비는 의도적으로 제외.
- **사내 매매 가격**: 시장가격(market price) 기준으로 아메바 리더 간 협상으로 결정. 내부 거래도 외부 거래와 동일하게 취급.
- **JAL 실증**: 2010년 도산 상태에서 아메바 경영 도입 후 첫 결산(2011년 3월)에서 영업이익 1,884억 엔 달성 — 세계 항공사 중 최고 수익.

## 1. 채산표 실제 양식 — 항목 구조

### 1-1. 기본 논리 (수직 구조)

채산표는 일반 손익계산서(P&L)와 다르다. "가계부처럼 쉽게 이해할 수 있도록" 설계된 것이 핵심 원칙이다.

```
[수입 측]
총출하(그로스 생산고)  =  사외출하 + 사내판매
                      ↑             ↑
                  외부 고객에게   다른 아메바에게
                  판매한 금액     판매한 금액

(-) 사내구매           =  다른 아메바에서 매입한 금액

= 총생산(네트 생산고)   =  해당 아메바가 진짜로 만들어낸 가치

(-) 경비(控除額)        =  원재료비, 외주가공비, 전화통신비,
                          여비교통비, 포장운임비 등 (노무비 제외)

= 차인매출(부가가치)    =  아메바가 창출한 순부가가치

[시간]
총시간                 =  해당 아메바 전원의 총 노동시간

[지표]
시간당 채산            =  차인매출 ÷ 총시간
```

- **확인 수준**: 확인됨
- **출처**: 東京メトロポリタン税理士法人 채산표 해설, 税理士法人エヴィス 교세라 채산표 설명
  [source: tm-tax.com/mailmag/zaimu/zaimu340/, e-zeirisi.com/kyocera-ameba-2-4272.html]

## 2. 부서(아메바) 단위 채산표 — 운영 구조

### 2-1. 아메바 분할 원칙

- 단위 크기: **5~50명** (교세라 기준 5~10명이 이상적)
- 분할 기준: 제품별, 공정별, 기능별, 수익책임별
- 아메바 수: 교세라는 1990년대에 700개 이상의 아메바로 운영

- **출처**: [korea.kyocera.com 아메바 경영](https://korea.kyocera.com/inamori/management/amoeba/index.html), [Wikipedia Amoeba Management](https://en.wikipedia.org/wiki/Amoeba_Management)
- **확인 수준**: 확인됨

### 2-2. 사내 매매(社内売買) 가격 결정 방식

가장 논쟁이 많은 부분. 확인된 원칙은 다음과 같다:

1. **기본 원칙: 시장가격(マーケットプライス)** — 내부 거래도 외부 시장 가격에 준함
2. **협상 방식**: 아메바 리더 간 직접 협상으로 결정. 본사가 일방적으로 정하지 않음
3. **최종 산출물 기준**: 최종 고객이 지불할 가격을 역산하여 내부 단계별 가격 설정
4. **시장 변동 반영**: 외부 시장 가격 변동이 내부 사내 매매 가격에 즉시 반영됨

예: 고객이 완제품에 100만 원을 지불 → 조립 아메바가 80만 원에 사내 판매 → 가공 아메바가 50만 원에 사내 판매 → 소재 아메바가 원가+마진으로 생산

- **출처**: [Kyocera Amoeba Management - Corporate Rebels](https://www.corporate-rebels.com/blog/amoeba-management), [global.kyocera.com Amoeba](https://global.kyocera.com/inamori/about/manager/amoeba/amoeba01.html)
- **확인 수준**: 확인됨 (시장가격 원칙), 가능성 (협상 세부 방법은 사내 기밀)

## 3. 노무비를 경비에 넣지 않는 이유 — 철학적 핵심

### 3-1. 표면적 이유: 통제 불가능성

> "노무비(인건비)는 회사의 채용 방침, 인사·총무 관련 방침에 의해 금액이 거의 결정된다. 따라서 아메바 책임자가 노무비를 통제하는 것은 불가능하다."

채산표는 아메바 리더가 실제로 통제 가능한 항목만 넣는 것이 원칙. 통제 불가 비용을 넣으면 평가가 왜곡된다.

- **출처**: [am-link.jp → KCCS 아메바경영 LINK](https://www.kccs.co.jp/consulting/topics/2025/001/), 동경메트로폴리탄 세무사법인
- **확인 수준**: 확인됨 (복수 소스에서 동일 설명 등장)

### 3-2. 철학적 이유: 인간을 비용으로 보지 않는다

이나모리의 핵심 발언:

> "사람을 인건비라는 경비로 보는 것이 아니라, 시간으로 노동생산성을 얼마나 늘려갈 수 있는가라는 관점으로 본다. 시간으로 표현함으로써 임금을 어떻게 줄일까가 아니라 생산성을 무한히 늘려가는 것을 목표로 한다."

즉:
- **비용 삭감 사고방식** (노무비를 경비에 넣음): "사람을 줄이면 이익이 난다"
- **이나모리식 사고방식** (시간으로 측정): "같은 사람이 더 많은 가치를 만들면 시간당 채산이 올라간다"

시간당 채산이 오르면 자연스럽게 팀원 스스로 "쓸모없는 시간을 줄이자"는 의식이 생긴다 — 위에서 강제하지 않아도.

- **출처**: [frontier-eyes.online 아메바경영](https://frontier-eyes.online/amoeba-management/), [KCCS 채산표 해설](https://www.kccs.co.jp/consulting/service/amoeba/program/profit/)
- **확인 수준**: 확인됨

## 4. 월간 운영 사이클 — 실무

### 4-1. 연간 → 월간 계획 연결 구조

```
[연초]
회사 마스터플랜(연간 목표) 수립
    ↓
각 아메바별 연간 목표 배분

[매월 반복 — 월간 PDCA]
Plan   → 각 아메바 리더가 다음 달 채산 계획표 작성
           (목표 매출, 목표 경비, 목표 시간당 채산)
Do     → 당월 실행
Check  → 월말 실적 채산표 작성 + 계획 대비 차이 분석
Action → 다음 달 채획에 반영
```

### 4-2. 월말 채산표 공개 원칙

- 월말 마감 후 **즉시 집계** (가능한 한 빨리 — 이상적으로 월초 1~3일 내)
- **전 사원에게 공개**: 모든 아메바의 채산표를 전직원이 볼 수 있음
- 이것이 "전원 참가 경영"의 핵심 메커니즘

### 4-3. 회의 운영

- **아메바 리더 회의**: 채산표 기반으로 전달/보고가 아닌 "분석과 대책" 논의
- "왜 시간당 채산이 목표에 못 미쳤는가" → 원인 분석 → 다음 달 실행 계획
- KCCS(교세라 컨설팅)는 이를 "보고회의에서 CAP회의(Check-Action-Plan)로의 전환"이라 표현

- **출처**: [KCCS 마스터플랜 컨설팅](https://www.kccs.co.jp/consulting/service/amoeba/program/masterplan/), [NDL 서치 — 아메바경영 월차PDCA](https://ndlsearch.ndl.go.jp/books/R000000004-I6682665)
- **확인 수준**: 확인됨

## 5. 실제 사례

### 5-1. 교세라 (창업 사례)

- 1963년 이나모리가 시간당 채산 제도 최초 고안
- 1965년 아메바 경영 시스템으로 정착
- 2000년대 기준 700개 이상 아메바로 운영
- 교세라 창업 이후 단 한 번도 연간 적자 없음 (수십 년간)

- **출처**: [global.kyocera.com 아카이브](https://global.kyocera.com/inamori/archive/episode/episode-15.html)

### 5-3. KDDI (통신 사업)

- 이나모리가 DDI(현 KDDI) 창업 시 동일한 아메바 방식 적용
- 사업부문별 독립 채산 → 일본 통신시장 3위 사업자로 성장
- **확인 수준**: 확인됨 (개략적 사실만, 상세 채산표 데이터는 비공개)

### 5-4. 한국 기업 사례

- **메리츠금융그룹**: 아메바 경영 원칙 도입 사례로 한국경제신문에서 언급됨. 팀 단위 독립 채산제 운영.
- **강성부 펀드(KCGI)**: 투자 대상 기업에 아메바 경영 원칙 요구한 것으로 보도됨
- 한국의 경우 일본 닛산계 기업(일본전산 등)에서 영향 받은 기업들이 간접 도입한 사례가 많음
- **확인 수준**: 가능성 (한국어 언론 보도 기반, 상세 채산표 운영 방식은 미확인)
- **출처**: [한국경제 — 한국 기업 아메바 경영](https://www.hankyung.com/article/202203284154i)

## 한계 / 미확인 항목

1. **사내 매매 가격 결정의 실제 분쟁 처리 방법**: 아메바 리더 간 협상이 결렬될 경우 상위 기관이 어떻게 개입하는지 공식 문서에서 확인 불가. 교세라 내부 규정으로 추정.
2. **한국 기업 채산표 실제 양식**: 메리츠금융 등 국내 도입 기업의 실제 채산표 양식은 기업 기밀로 공개 자료 없음.
3. **판매부문 아메바 채산표 구조**: 이번 리서치는 제조부문 중심. 판매/서비스 아메바의 경우 "총출하 = 사외출하만" (사내판매 없음)이 일반적이나 세부 구조는 추가 확인 필요.
4. **시간 산정 방법**: 잔업 시간 포함 여부, 관리직 시간 계산 방법 등은 기업마다 상이하며 공식 가이드라인 미확인.
