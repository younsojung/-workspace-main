# 이나모리 가즈오의 채산표 — 시간당 부가가치로 일과 삶을 재설계하다

> 윤소정의 기획법(8단계)으로 설계됨
> 작성일: 2026-05-19
> 기획자: 윤소정

---

## 한 줄 컨셉

> **"내 시간에 가격을 매겨본 적 있는가?"**
> 회사의 채산표를 배우는 동안, 동시에 자기 인생의 채산표 한 장이 손에 잡힌다.

---

## 메타

| 항목 | 내용 |
|------|------|
| **대상** | 직장인(5~15년차) · 1인사업자/프리랜서 · 작은 조직 리더 (15~20명) *(추정)* |
| **사전 지식** | 회계 비전공자. 손익계산서 들어봤지만 직접 짜본 적 없음 *(추정)* |
| **시간 구조** | **단원 모드** · 4회 × 120분 (총 8시간) · 주 1회 4주 |
| **형식** | 하이브리드 — 오프라인 워크샵 중심 + 매주 사이 온라인 과제 *(추정)* |
| **공간** | 이동 가능한 워크샵 룸 (테이블 4~5개, 화이트보드 2개) |
| **사용 도구** | 노트북 · 채산표 워크시트(A4) · 본인 가계부/통장 데이터 · 인스타 또는 본인 채널 |
| **운영 모드** | 단원 (권장) |

---

## 학습 목표 (4 차원)

> 8단계 사이클의 네 사분면에 대응.

- **왜 배우는가**: "바쁘게 살고 있는데 왜 통장은 안 부푸는가" — 자기 시간이 가치로 환산되지 않는 구조를 직시한다
- **무엇을 배우는가**: 이나모리 가즈오의 채산표(부가가치 ÷ 노동시간) 원리. 시간당 부가가치 = (수입 - 인건비 제외 경비) ÷ 총노동시간
- **어떻게 쓰는가**: 회사 채산표 1장 + 본인 개인 채산표 1장을 직접 작성한다
- **무엇이 새로 생기는가**: "내 시간의 가격"이라는 새로운 단위를 가지고 일·돈·관계를 다시 본다

**전환점 1순위**: **행동** — 끝나고 본인의 채산표 한 장이 손에 들려 있어야 한다.

## 단원 흐름 (4주)

```
1주차 ─ 만남 + 알아챔        ─ "내 시간은 얼마짜리인가"
2주차 ─ 상상 + 정보           ─ "이나모리는 무엇을 발견했나"
3주차 ─ 연습 + 나의 것       ─ "회사 채산표 → 나의 채산표"
4주차 ─ 다듬기 + 펼치기      ─ "이걸 가지고 나가는 한 사람의 결심"
```

## 8단계 기획

### 1. 만남 — *너의 한 시간은 얼마였는가*

> **핵심 질문**: "지난 일주일에서, 가장 가치 있었던 1시간과 가장 허비된 1시간을 적어봐. 각각 얼마짜리였나?"

| 항목 | 내용 |
|------|------|
| **학습자 활동** | (10분) 개인 글쓰기: 가장 가치 있던 1시간 + 가장 허비된 1시간 + 본인이 매긴 가격 / (10분) 4인 1조 나눔: "왜 그 가격을 매겼는가" / (5분) 전체 공유: 가장 높은 가격, 가장 낮은 가격 |
| **기획자 역할** | 동기 부여자 — 본인 이야기 먼저 공개. "나는 작년에 강의 1시간에 50만원을 매겼는데, 그 강의 준비에 10시간이 들었더라. 그러면 진짜 내 시간당은 5만원이었다." 절대 *정답*을 던지지 않음 |
| **사용 도구** | A4 종이 1장 · 본인 일주일 캘린더 (각자 폰으로 켜기) · 윤소정 본인 사례 슬라이드 1장 |
| **시간** | **25분** (전체 120분의 약 21% — 만남 비중을 상한으로. 성인 학습자라 *왜 지금* 비중 ↑) |
| **성공 신호** | 학습자가 "어 잠깐, 그러고 보니까…" 하고 자기 일주일을 *다시 들여다보기 시작*. 누군가 "내가 시간을 너무 싸게 팔고 있었네"라고 말하면 성공 |

### 2. 알아챔 — *우리는 왜 시간을 가격으로 안 봤을까*

> **핵심 질문**: "왜 우리는 월급을 '월'로 받고, 시간으로 받지 않았을까? 그게 우리에게 무엇을 가렸을까?"

| 항목 | 내용 |
|------|------|
| **학습자 활동** | (10분) 1단계에서 쓴 가격을 *시급*으로 환산하고 본인 월급 시급과 비교 / (10분) 4인 1조: "내 시간에 가격을 안 매기게 만든 환경 3개" 찾기 (예: 월급제, 정시 퇴근 문화, "내 시간은 공짜라는" 가정) / (5분) 패턴 명명하기 — "시간을 안 보이게 만드는 것들" |
| **기획자 역할** | 안내자 — "그게 너만의 경험인가? 옆 사람도 같은가?" 같은 질문으로 개별→패턴 이행을 도움. 가치 판단 자제 |
| **사용 도구** | 화이트보드(패턴 그래프) · 시급 계산 미니 워크시트 (월급 ÷ 월 노동시간) |
| **시간** | **25분** (약 21%) |
| **성공 신호** | "우리는 시간을 단위로 안 다뤘구나"라는 공통 자각. 화이트보드에 "시간을 가린 것들" 리스트 5개 이상 |

### 3. 상상 — *교세라의 한 공장에서 무슨 일이 있었나*

> **핵심 질문**: "1963년, 27살 이나모리가 두 공장 중 어느 쪽이 더 효율적인지 어떻게 알아냈을까?"

| 항목 | 내용 |
|------|------|
| **학습자 활동** | (10분) 1963년 시가 공장 스토리 듣기 — 교토 본사와 신설 시가 공장을 어떻게 비교할까 / (15분) 4인 1조 마인드맵: "두 공장을 같은 자로 재는 방법" 자유 토론 → 자기들 식 공식 만들어보기 / (5분) 그룹별 공식 발표 → 화이트보드에 나란히 |
| **기획자 역할** | 큐레이터 — 1963년 시가 공장 에피소드 *이야기로* 들려주기. 채산표라는 단어는 아직 안 꺼냄. 본인들이 *발견하는* 자리 만들기 |
| **사용 도구** | 큰 화이트보드 · 사진 (이나모리·교세라 시가 공장 흑백 사진) · 컬러 마커 |
| **시간** | **15분** (약 13%) |
| **성공 신호** | 한 그룹이라도 "수입 빼기 비용 나누기 시간"에 도달하면 대성공. 못 도달해도 *시도 자체*가 다음 단계의 흡수율을 결정 |

### 4. 정보 — *채산표의 해부학*

> **핵심 질문**: "이나모리가 만든 채산표는 정확히 무엇인가, 일반 손익계산서와 무엇이 다른가?"

| 항목 | 내용 |
|------|------|
| **학습자 활동** | (15분) 미니 강의 1: 채산표의 5가지 핵심 — ①시간당 부가가치 공식 ②인건비를 경비에서 빼는 이유 ③아메바(5~50명) 단위 ④사내 매매 시장가격 원칙 ⑤월 1회 전원 공개 / (10분) 미니 강의 2: 회계학 7원칙 (캐시 베이스, 일대일 대응, 근육질, 완벽주의, 더블체크, 채산 향상, 투명 경영) / (10분) 실 사례 — JAL 도산→2년 8개월 만에 영업이익 1,884억엔 |
| **기획자 역할** | 전문가/강의자 — 슬라이드는 *최대 10장*. 숫자 1개 = 슬라이드 1장 원칙. 일대일 대응·시간당 채산은 *반드시 손글씨로 칠판에* |
| **사용 도구** | 슬라이드 (실 채산표 예시 표 포함 — 사외출하 35,000 / 사내판매 25,000 / … / 시간당 채산 5,000엔/H) · JAL 재건 인포그래픽 1장 |
| **시간** | **30분** (약 25% — 가장 긴 단계, 하지만 25% 상한 준수) |
| **성공 신호** | 학습자가 칠판에 그려진 채산표 공식을 *자기 손으로* 옮겨 적고 있음. "왜 인건비를 뺐어요?"라는 질문이 나오면 만점 |

### 5. 연습 — *교세라 가상 부서로 채산표 1장 만들기*

> **핵심 질문**: "주어진 데이터로, 이 부서의 시간당 채산을 직접 계산해보자"

| 항목 | 내용 |
|------|------|
| **학습자 활동** | (5분) 워크시트 배포 — 가상 부서 "A아메바" 한 달 데이터 (사외매출, 사내매출, 사내구매, 원재료비, 외주비, 통신비, 노동시간) / (20분) 개인별 채산표 완성 → 시간당 채산 산출 / (10분) 2인 1조 비교 → 다른 사람과 결과 같은지 확인 → 차이 나면 어디서 났는지 추적 |
| **기획자 역할** | 코치 — 돌아다니며 막힌 사람 도움. *정답*을 알려주지 않고 *어느 줄에서 막혔는지*를 묻는다. 오류를 환영 ("그 실수가 좋은 실수예요") |
| **사용 도구** | A아메바 워크시트(A4 1장) · 계산기 또는 노트북 · 정답표는 마지막에만 공개 |
| **시간** | **20분** (약 17%) |
| **성공 신호** | 전원이 시간당 채산 숫자 1개를 본인 손으로 산출. 정답 일치 여부보다 *공식을 자기 손으로 통과한 것*이 핵심 |

### 6. 나의 것 — *나의 채산표 한 장* ⭐

> **핵심 질문**: "회사의 채산표를 이제 나에게 적용한다면, 너의 한 달은 어떤 표가 되는가?"
> ⚡ **이 단계부터 기획자 권한이 학습자에게 이양된다.**

| 항목 | 내용 |
|------|------|
| **학습자 활동** | (5분) 개인 채산표 양식 받기 — 빈 칸뿐. 행·열 직접 정의 / (15분) 본인 통장·가계부·캘린더 데이터로 한 달 본인 채산표 완성. 행 항목 자율 선택 (본업/부업 수입, 시간 투입, "사내구매" 격인 외부 의존, 경비, 시간당 부가가치) / (10분) 4인 1조 발표: "내 채산표에서 가장 충격적인 줄 하나"만 공유 |
| **기획자 역할** | 촉진자 — *덜 가르치고 더 묻기*. "왜 그 항목을 넣었어요?" "그 시간에 더 넣어야 할 게 있다면요?" 양식을 *처방*하지 않음 |
| **사용 도구** | 개인 채산표 워크시트 (반쯤만 채워진 양식) · 본인 폰의 가계부 앱 · 캘린더 / *(자료 부족 시 대안)*: 지난 한 달 추정치로 적되 4주차까지 보강 과제 |
| **시간** | **30분** (약 25% — 학습자 주도 단계가 단원의 중심. 일반 비중보다 늘려 잡음) |
| **성공 신호** | 학습자별로 *서로 다른 모양*의 표가 나옴. 누군가 "내 시간당 부가가치가 시급 알바보다 낮네"라고 말하면 그게 *행동 변화 트리거* |

### 7. 다듬기 — *내 채산표는 정직한가*

> **핵심 질문**: "이 채산표가 정말 너의 한 달을 정직하게 비추고 있는가? 무엇을 가렸는가?"

| 항목 | 내용 |
|------|------|
| **학습자 활동** | (10분) 자기 표 셀프 비판 — "이 표가 안 보여주는 것 3개" 적기 (감정 노동, 무급 돌봄, 친구와의 시간 등) / (10분) 2인 1조 동료 피드백 — "이 표를 진짜로 만들려면 추가해야 할 한 줄" 추천 / (5분) 채산표 v2로 1줄 수정 |
| **기획자 역할** | 비판적 동료 — "이 표 안에 너의 *살아있음*이 들어가 있는가?" 같은 질문 던지기. 짓누르지 않게 — 더 높은 기준은 *살짝* 제시 |
| **사용 도구** | 동료 피드백 양식 (1장, 3칸: 좋은 점/안 보이는 것/한 줄 제안) |
| **시간** | **15분** (약 13%) |
| **성공 신호** | 본인 채산표가 단순 가계부가 아니라 *본인 인생 측정 도구*로 자라기 시작 |

### 8. 펼치기 — *한 사람의 결심을 세상에 한 줄로*

> **핵심 질문**: "이 채산표로 다음 한 달, 너는 *무엇을 안 할 것인가*?"
> 🎯 **청중**: 본인의 인스타·블로그·뉴스레터 구독자 또는 본 클래스 단톡방의 다른 동기들. *반드시 외부 공개*.

| 항목 | 내용 |
|------|------|
| **학습자 활동** | (10분) 본인 채산표를 사진 1장으로 정리 + 1줄 결심: "내 시간당 부가가치를 ___까지 올리기 위해, 이번 달 ___을 안 한다" / (5분) 단톡방 또는 SNS 게시 / (10분) 단원 전체 회고 — 4주 전 본인과 지금 본인의 차이 발표(1인 1분) |
| **기획자 역할** | 청중·연결자 — 본인이 모든 게시물에 댓글 1개씩 달아주기. 다음 사이클(3개월 후 후속 워크샵) 연결 제시. *의도적으로 뒤로 물러남* |
| **사용 도구** | 인스타그램 / 블로그 / 본 클래스 단톡방 / 본인 이메일 뉴스레터 |
| **시간** | **15분** (약 13%) |
| **성공 신호** | 본인 외 *최소 한 명의 외부 청중*이 그 게시물에 반응함. 학습이 교실 밖으로 나감 |

## 평가 / 성취 확인

> 학습이 일어났는지 어떻게 확인하는가?

- **단계별 형성 평가** (학습 중):
  - 5단계 직후 — 전원이 A아메바의 시간당 채산 숫자를 본인 손으로 계산했는가
  - 6단계 산출물 — 본인 채산표에 *서로 다른 모양*의 표가 나왔는가 (똑같은 양식이면 실패)
- **최종 평가** (학습 후):
  - **행동(1순위)** 평가: 본인 채산표 1장이 본인 인스타/블로그/단톡방에 *실제로 게시되었는가*
  - 4주 후 후속: 한 달 동안 본인 채산표를 *다시 한 번이라도* 갱신했는가
- **공유 / 펼치기 청중의 반응**:
  - 학습자별 게시물에 외부 반응(좋아요/댓글) 개수
  - 단톡방에서 *다른 학습자에게* 이 개념을 설명한 사례 발생 여부

## 단원 운영 핵심 원리 — 윤소정의 손글씨 메모

1. **만남(1단계) 절대 짧게 처리 금지** — 시간당 부가가치라는 *낯선 단위*를 받아들이려면 본인 일주일을 정직하게 들여다보는 시간이 먼저다.
2. **4단계(정보)를 부풀리지 말 것** — 채산표 공식은 한 줄. 30분 안에 다 들어간다. 30분을 안 넘긴다. 욕심내면 5/6단계가 죽는다.
3. **6단계가 이 수업의 심장** — 회사 채산표는 *재료*다. 본인 채산표가 만들어지지 않으면 4주 전체가 강의가 된다.
4. **8단계 청중은 반드시 *교실 밖*** — 단톡방 안에서만 공유하면 펼치기가 무너진다. 본인 SNS·뉴스레터·블로그 중 *공개 채널 하나*를 반드시 거치게.
5. **3개월 후 후속 모임을 미리 약속** — 단원 종료 시점에 3개월 후 1회 후속 워크샵 일정을 캘린더에 박아둔다. 다음 사이클의 씨앗.
