# 사례 패턴 — 기획서 자동 생성 시 적용 규칙

> 외부 리서치(111개 사례 + 75개 소스)에서 추출한 핵심 패턴.
> 8단계 기획서를 자동 생성할 때 이 규칙들을 *반드시* 적용한다.

---

## 패턴 1 — Step 1(만남)의 질이 전체 성패를 결정한다

효과적이었던 사례들은 *예외 없이* 만남 단계에서 학습자의 실제 삶·경험·감정을 구체적으로 건드리는 접속점을 설계했다. 추상적 "관련성 소개"가 아니다.

**적용 규칙**:
- 만남 단계에 **항상 셋 중 하나**를 배치:
  1. **개인 이야기 공유** (자기 경험 떠올리기 / 짝과 나누기 / 글쓰기)
  2. **도발적 질문** (정답 없음 / 양극화된 견해 / 흔한 가정 뒤집기)
  3. **감각적 자극** (영상 / 사진 / 음악 / 실물 시연)
- 절대 금지: "오늘은 X에 대해 배우겠습니다" 같은 *형식적 도입*
- 시간 비중이 *짧아도* 강도는 강해야 함

---

## 패턴 2 — 8단계의 진짜 메커니즘은 "기획자 역할의 점진적 이양"

학습 효과가 높았던 사례의 공통 구조:
- Q1~Q2 (1~4단계): 기획자가 *주도* (동기 부여, 안내, 강의)
- Q3 (5~6단계): 기획자가 *코치/촉진자*로 전환
- Q4 (7~8단계): 학습자가 *주도*, 기획자는 *청중/연결자*

**적용 규칙**:
- 각 단계 *기획자 역할*을 명시하고, **5단계 → 6단계에서 권한 손바뀜이 분명하게** 보이도록 활동 설계
- 6단계 활동에는 항상 "*자율적*", "*자기 맥락*", "*창의적 변형*" 같은 단어 등장
- 8단계 활동은 *학습자 주도* 형식 (발표, 가르치기, 공유). 기획자 강의 X

---

## 패턴 3 — 단원(2~4주) 단위가 한 차시보다 압도적으로 강력

효과가 확인된 30여 개 독립 연구에서 대부분 *단원 단위* 운영. 한 차시에 8단계를 우겨넣으면 각 단계 10분도 안 됨.

**적용 규칙**:
- 사용자가 *총 시간*만 알려주면 적정 운영 모드 추천:
  - 60~90분 → 한 차시 모드 (각 단계 짧게)
  - 180분 이상 또는 다회차 → 단원 모드 *권장*
- 단원 모드일 때: 각 단계에 1~2차시 할당. 1단계는 1차시 전체를 줄 수도 있음
- 한 차시 모드일 때: 결과물 끝에 "단원 모드 권장" 한 줄 첨부

---

## 패턴 4 — 성인 학습자에게 "Why 먼저"가 결정적

기업 교육에서 4MAT이 빠르게 채택된 이유. 성인은 *왜 배워야 하는지* 납득되지 않으면 학습 참여를 거부한다.

**적용 규칙**:
- 학습자가 성인/직장인이면:
  - 1단계 시간 비중을 *상한*(15%)으로
  - 4단계(정보) 전에 *"왜 지금, 왜 너에게"*를 반복적으로 짚는 활동 추가
  - 6단계 활동을 *자기 업무 적용*으로 강제

---

## 패턴 5 — 이론적 비판에도 불구하고 실용성은 별개로 유효

좌우뇌 이분법은 현대 신경과학에서 비판받지만, 수업 설계 도구로서의 효과는 30+ 독립 연구에서 반복 확인됐다. *왜 효과가 나는지*의 메커니즘은 다음 3개일 가능성이 높다:

1. **다양한 학습 스타일 포용** — 한 사이클에 4가지 유형 모두 자기 자리 있음
2. **반복 강화** — 같은 개념을 *다른 모드*로 4~5번 만남
3. **점진적 깊이** — 표면(만남) → 분석(알아챔) → 그림(상상) → 정보(정보) → 실행(연습) → 자기화(나의 것) → 정제(다듬기) → 통합(펼치기)

**적용 규칙**:
- 8단계 사이에 *같은 개념이 다른 형태로 반복*되는지 점검
- 각 단계에서 학습자가 *서로 다른 능력*을 쓰는지 점검 (정서/분석/시각/언어/실행/창의/평가/공유)

## 추가 미세 패턴

### 패턴 6 — 1단계는 *수업 시작 전*에 시작될 수 있다

일부 효과적 사례는 1단계를 *사전 과제*로 보냄 (영상 시청, 짧은 질문에 답하기). 본 수업 시작 시점에 이미 *만남*이 일어나 있음.

**적용 가능**: 시리즈/단원 모드에서 적극 활용

### 패턴 7 — 4단계(정보)는 *전문가 게스트*가 들어오면 강해진다

학습자가 *기획자 외 인물*의 목소리를 들으면 정보 흡수율이 올라감.

**적용 가능**: 사용자가 게스트 가능하다고 하면 4단계에 배치 추천

### 패턴 8 — 8단계(펼치기)는 *물리적 청중*이 있어야 작동

발표가 *벽을 향해* 이뤄지면 8단계가 무너짐. 누가 듣는지가 중요.

**적용 규칙**:
- 8단계에 항상 *청중 명시*: "누가 듣는가? 어디에 게시되는가?"
- 청중이 없는 학급/그룹이라면 *외부 청중* 만들기: SNS, 학교 신문, 블로그, 다른 반

---

## 자동 생성 체크리스트

기획서 생성 직후 다음을 확인:

- [ ] 만남(1단계)에 셋 중 하나 들어갔나? (개인 이야기 / 도발 질문 / 감각 자극)
- [ ] 5→6단계에서 *권한 손바뀜*이 명확한가?
- [ ] 6단계에 *자기 맥락 적용* 활동이 있나?
- [ ] 8단계에 *청중*이 명시되어 있나?
- [ ] 시간 비중이 [10-15 / 10 / 10-15 / 20-25 / 15-20 / 10-15 / 10 / 10-15] 범위인가?
- [ ] 성인 학습자라면 1단계 *상한*까지 갔나?
- [ ] 한 차시 모드라면 *단원 권장* 메모 첨부됐나?

미충족 항목 있으면 *생성 후 자체 수정* 또는 사용자에게 *이 부분만 다시 확인*해달라고 요청.
