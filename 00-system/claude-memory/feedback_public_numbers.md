**활성 구독자 정확 숫자(예: 5,412명)는 외부 카피에 사용하지 않는다.**

**Why:** 활성 구독자 수는 비공개. 내부 진단/대시보드용으로만 쓰는 숫자다. 사이트·인스타·유튜브 등 외부 노출 카피에 이 숫자가 들어가면 안 된다.

**How to apply:** 사이트 카피, 광고 카피, 인스타·유튜브 캡션, 외부 홍보물 작성 시 — 5,412명을 비롯한 *활성 구독자 정확 수치*는 빼고 작성. 외부 공개 가능한 대체 숫자(예: "14년", "매월 1권", "80편 발행", "100% 오가닉", "유튜브 20만 / 인스타 8만" 등 — 단, 각 숫자도 사용 전 작가에게 외부 공개 가능 여부를 한 번 더 확인)로 갈음한다.
