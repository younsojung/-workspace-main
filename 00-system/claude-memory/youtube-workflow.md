윤소정 유튜브 영상 작업 5단계 — 검증 영상 3편 (10만+) 분석 후 정립.

1. **한 달 정리** — Raw material 5-10개 (영상 주제 후보)
2. **컨셉 박기** — 자기 폭로 + 핵심 단어 + 권위 인물
3. **형식 결정** — 인터뷰 (30-47분) / 자기 정리 (14분) / 혼합
4. **스크립트 작성** — 5단계 공식: HOOK · PROOF · FRAMES · CRY · CALL
5. **자료·촬영 가이드** — 노트·책·비밀라이브·댓글보상 + 컷·음악·자막

**스크립트 5단계 공식**:
- HOOK (0:00-0:30): 자기 폭로 1줄 + 핵심 단어 + [음악]
- PROOF (0:30-3:00): 시청자 호명 + 자기 한 달 + 권위 위임 (5개+ 자격)
- FRAMES (3:00-7:00): 3분법 (이분법·삼분법) — 4개 이상 ❌
- CRY (7:00-10:00): "그때 알았더라면 그렇게 ___하지 않았을 것" 자기 폭로 클라이맥스
- CALL (10:00-끝): 오늘 한 가지 + 자료 + 비밀 라이브 + 댓글 보상 (다섯 명 책 선물)

**Why:** 사용자가 81호 슈퍼샘플 유튜브 작업 중 검증된 윤소정 영상 3편 패턴 분석 → 5단계 워크플로우 + 5단계 공식 함께 박음 (2026-05-25).

**How to apply:**
1. 사용자가 "유튜브 영상 시작" / "이번 영상" / "다음 영상" / "유튜브 스크립트" 등으로 진입하면 *반드시 1단계부터*
2. 형식·길이는 *권위 인물 있으면 인터뷰 (30분), 없으면 자기 정리 (14분)* 기본
3. **끝점부터 그린다** — 시청자가 *무엇을 할 사람*이 될지 (CALL)를 박고 역으로 CRY·FRAMES·PROOF·HOOK
4. 시그니처 어휘 반드시: "근데+진짜" 콤보 / "있거든·그지·잖아" / "여러분·친구들·얘들아" / 시각 비유 / [음악] 컷
5. 외부 공개 숫자 가드 — 활성 구독자·매출 수치 ❌
