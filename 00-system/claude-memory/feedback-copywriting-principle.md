윤소정의 카피·CTA·hero copy 작업 시 3원칙:

1. **직관적** — 한 번에 이해. 추상·은유 ❌
2. **실용적** — 받았을 때 *뭘 하는지* 즉시 보임
3. **누구나 알아볼 수 있어야** — 윤소정 팬·내부자만 이해되는 어휘 ❌

**Why:** 2026-05-23 사이트 개편 작업 중. 사용자가 메인 CTA를 *감성적*이게 요청 → 내가 *시적·추상*(예: "이 책, 곁에 두기", "한 페이지 펼치기", "이 페이스에 스며들기")으로 잘못 해석. 사용자: "다 0점짜리야. 직관적이고 실용적이여야해. 누가 봐도 알아볼 수 있어야해."

윤소정의 *감성* ≠ 시적/문학적. 윤소정의 감성 = **친밀한 직설** (예: "안녕, 영혼의 친구. 이거 읽어봐", "이번 달 내가 공부한 거 다 적어뒀어, 따라해봐").

**How to apply:**
1. 카피 후보 던지기 전, "윤소정 모르는 사람이 봐도 *무엇을 하는지* 즉시 이해되나?" 자가 검증
2. 동사: *만나다·스며들다·곁에두다·펼치다* 같은 시적 동사보다 → *읽다·받다·시작하다·뛰다·따라하다* 같은 직설 동사 우선
3. CTA는 *행동*이 명확해야 — "이 책 곁에" ❌ → "이 책 읽기" ✓
4. 감성 ≠ 추상. 감성을 원하면 *친밀 호칭/1인칭/말투*로 살림 (예: "같이", "내가", "너")
5. 사용자의 실제 영상·콘텐츠 어휘에서 단어 가져오기 (생각구독 corpus 80호 = 기준 보이스)
6. 관련: [[생각구독-corpus]] (voice_reference), [[feedback-definition-consent]] (정의는 사용자가 박는다)
