윤소정의 디자인/타이포 기준은 **비넬리(Massimo Vignelli) 원칙**이다 (카드뉴스 "부크크명조 + 비넬리" 와 동일 기준).

- 크기는 소수만 (작은/본문/중간/큰 4단계).
- 단계 사이는 일정한 비율(1.25배). **가장 큰 글씨도 본문의 2배 이내.**
- 강조는 크기가 아니라 **굵기(weight)·여백·위치**로.

**Why:** "강조하는 글씨가 너무 크다, 비넬리 원칙을 잘못 배운 것 같다"고 직접 교정함 (2026-05-24). 디스플레이를 88px(clamp 5.5rem)까지 키운 게 비넬리 위반이었음.

**How to apply:** 사이트·카드뉴스·핸드아웃 등 모든 비주얼 작업에서 강조하려고 글씨를 키우지 말 것. 한 단계만 올리고 굵게 하거나 둘레 여백을 비운다. 타입 스케일은 1.25 비율, 디스플레이 상한 ~31px(2rem). 관련: [[feedback-copywriting-principle]] (직관·실용 = 글에서, 절제 = 디자인에서).
