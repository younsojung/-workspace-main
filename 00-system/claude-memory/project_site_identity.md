**younsojung.co.kr 사이트의 정체성 = e-book 판매 사이트.**

**Why:** 작가가 2026-05-23에 사이트 카피 작업 중 명확히 결정. 사이트의 진짜 매출원은 80편 e-book 카탈로그(월 2,574만원, 전환율 4.24% — 이커머스 상위 10%). 생각구독은 외부 폼·SMS 시스템에서 독립적으로 운영되며, 사이트와 결제 시스템·결제 동선이 분리되어 있다. 사이트 = 생각구독 깔때기로 재정의하라는 진단 권고가 있었지만, 작가가 분리 유지하기로 결정.

**How to apply:**
- 사이트 카피·구조 작업 시 메인 동선은 *e-book 판매* (단권 13,000원 / 연도세트 140,400원 / 풀세트 75권 780,000원)
- 생각구독은 보조 메뉴로 유지 (메뉴 7번째 위치 OK)
- 사이트 KPI = e-book 매출 / 단권→풀세트 업셀 / 재구매율
- 생각구독 재구독률 10% → 20% 작업은 **별도 트랙** (외부 폼 시스템 안에서)
- trus-frame Q1 "매월 1만명"의 두 트랙: ① 사이트(e-book 판매) ② 생각구독(재구독률 끌어올림). 카피 작업은 트랙 ①에 집중.
