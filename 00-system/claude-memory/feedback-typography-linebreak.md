**윤소정식 폰트 디자인의 핵심은 "잘 읽히게 하는 것"** — 잘 읽힌다 = 소리 내어 읽었을 때 자연스럽다.

- 텍스트를 **소리 내어 읽어보면** 띄어야 할 곳과 내려써야 할 곳이 있다.
- **숨 쉬는 타이밍에 아래로 내려쓴다(줄바꿈).** 그래야 글씨가 눈에 잘 들어온다.
- 띄어쓰기(공백)와 내려쓰기(`<br>`)를 **의도적으로 설계** — 화면 폭에 맡긴 기계적 줄바꿈 ❌.
- 예: ❌"선생님이 수업 전에 / 하는 것" → ✅"선생님이 / 수업 전에 하는 것" (주어 뒤가 숨 쉬는 자리).
- **한 호흡 = 한 줄**: 한 번에 쭉 읽히는 구절은 쪼개지 말고 붙여 둔다.
- **적용 범위**: 호흡 줄바꿈은 *짧은 핵심 문장·리드·제목·인용*에만. **길게 설명하는 "그냥 본문" 단락은 붙여서 흐르게 둔다**(억지 줄바꿈 ❌, keep-all로 단어만 보호).
- **정렬**: 붙여 쓰는 본문 단락 = 양쪽정렬(justify) + `word-break:normal`(어절 나뉘어도 빽빽한 블록, 책·신문식). 짧은 문장·호흡 줄바꿈 = 왼쪽정렬 + `keep-all`(단어 보호). keep-all은 왼쪽정렬에서만 필요, 양쪽정렬 본문엔 normal.
- **고아(orphan) 금지**: 조사(는/을/를)·단일 음절·문장 끝 글자가 줄 끝에 혼자 떨어지면 안 됨. `word-break:keep-all` 필수 + `text-wrap:pretty`. 그래도 남으면 줄바꿈 위치 조정하거나 **폰트 크기까지 조절해서 완성도 유지** (2026-05-24 추가 지적).

**Why:** 사용자가 직접 "윤소정식 폰트 디자인의 생명은 잘 읽힐 수 있도록 띄어쓰기와 내려쓰기를 잘 하는 것"이라고 명시 (2026-05-24).

**How to apply:** 사이트·카드뉴스·핸드아웃 등 모든 텍스트 배치 시, 줄바꿈을 폭에 맡기지 말고 소리 내어 읽어 호흡 단위로 `<br>`를 직접 넣는다. 한 문장도 호흡 단위로 나누면 리듬이 생긴다. 폰트 선택은 [[feedback-typography-vignelli]] + 제목/라벨=Pretendard, 흐르는 본문 단락=명조. 사이트 가이드: 10-projects/younsojung-site-renewal/design/design-guide.md (§1-B).
