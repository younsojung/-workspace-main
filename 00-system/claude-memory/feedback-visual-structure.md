윤소정은 선택지·작업 구조·비교·진단을 **표/트리/다이어그램 등 "그림"으로 시각화**해서 보여주면 훨씬 빠르고 정확하게 판단한다. 글로 길게 나열하지 않는다.

**Why:** 본인이 직접 "이런 식으로 그림으로 구조를 그려서 보여주니까 판단하기 쉽다"고 확인(2026-05-25). 시각 사고형.

**How to apply:** 옵션 비교·현황 정리·구조 설계·진단 결과는 표나 ASCII 다이어그램으로 먼저 그린다. AskUserQuestion의 preview도 적극 활용(실제 결과물 모양을 보여주면 선택이 쉬움). [[communication-style]] · [[eight-bars-training]]
