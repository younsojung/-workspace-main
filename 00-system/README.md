# 00-system

> 시스템 설정, 템플릿, 가이드. 워크스페이스 운영의 뿌리.

## 구조

```
00-system/
├── 01-templates/  # 기본 노트 템플릿 (Daily, Weekly, Project)
├── 02-scripts/    # (선택) 자동화 스크립트
├── 03-guides/     # 가이드 문서 (Claude Code, Prompt, Todo 등)
└── 04-docs/       # 참고 문서 (Johnny Decimal 가이드 등)
```

## 용도

- **01-templates/**: 새 노트 생성 시 복사해서 사용하는 기본 형식
- **02-scripts/**: npm scripts, Python 헬퍼 등 (선택 사용)
- **03-guides/**: 사용법 가이드 (Claude Code 활용, 프롬프트 작성 등)
- **04-docs/**: 시스템 이해를 돕는 참고 자료

## 수정 가능

자신의 업무에 맞게 템플릿과 가이드를 **자유롭게 수정**하세요.
