# 20-operations

> **지속적 운영 업무.** 종료일 없이 반복적으로 수행.

## 용도

- 체크리스트, SOP, 반복 프로세스
- 팀 운영, 고객 관리, 재무 등
- 월간/주간 리듬이 있는 업무

## 네이밍

- 21~29 사이 숫자 사용
- 예: `21-hr`, `22-customer-support`, `23-finance`, `24-content-ops`

## 10-projects와의 차이

| | 10-projects | 20-operations |
|---|---|---|
| 종료일 | 있음 | 없음 |
| 이동 | 완료시 archive | 계속 유지 |
| 예시 | 신제품 론칭 | 고객 응대 프로세스 |

혼동될 때 기준: **"이게 완료되는 시점이 있나?"**
- Yes → projects
- No → operations
