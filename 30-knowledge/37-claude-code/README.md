# 37-claude-code

> Claude Code 활용 관련 지식 — 공식 문서 + 실전 노트 + 자기 코드(skills/agents) 설계 기록.

## 하위 구조

| 폴더 | 용도 |
|------|------|
| [37.00-official-docs/](37.00-official-docs/) | 공식 문서 요약본 (Source of Truth). Skills/Subagents/Plugins/CLAUDE.md/Rules |

`37.01-learning`, `37.02-system` 같은 추가 폴더는 본인이 학습·구축 과정에서 생기는 자료에 맞춰 **필요할 때 만들어** 쓰면 된다. 템플릿이 강제는 아니다.

## 범위

- Claude Code 기능/워크플로우 노트
- 내가 만든 스킬/에이전트 설계 패턴
- 단축키, 설정, 통합 팁
- 하위 도구(MCP, Hooks, Plugins) 실전 기록

## 00-wiki와의 관계

- 여기: **출처와 맥락이 명확한 실전 기록**. 공식 문서 요약, 내 프로젝트에서의 적용, 특정 버전에서의 동작
- 00-wiki: **복리로 축적되는 개념·원칙**. "X란 무엇인가"를 물었을 때 답이 되는 토픽 페이지

범용적으로 재사용될 개념이 생기면 `wiki-ingest` 스킬로 `30-knowledge/00-wiki/`에 승격시킨다.
