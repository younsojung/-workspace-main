## 메인 ↔ 부제 조합 안티패턴

봇이 컨셉 잡을 때 **피해야 할 조합**:
- 메인이 ‘한 단어 정의’(예: 안목)인데 부제가 단순 동어반복 ("안목에 관하여") → **새 정보 없음**
- 메인이 ‘역설’(예: 느림의 정확도)인데 부제가 또 역설을 반복 → **메시지 평탄화**
- 메인이 너무 추상(예: 진심의 연결)이고 부제도 추상 → **둘 다 공중부양**

## 좋은 조합의 공통 규칙

1. **메인이 추상이면 부제는 구체 장면**으로 — 예: 진심의 연결(추상) + "데스커 라운지 3시 28분"(구체)
2. **메인이 후킹이면 부제는 풀어주기** — 예: 너의 악마성(후킹) + "욕먹는 나를 벌거벗기고 나서야" (풀이)
3. **메인이 짧으면 부제는 길게** — 메인 5자 이하면 부제는 15~25자
4. **둘 다 짧으면 둘 다 강해야** — 예: 시간의 세례 + "이제 날아갈 시간"
