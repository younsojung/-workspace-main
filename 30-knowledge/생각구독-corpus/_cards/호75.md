## 챕터 목록
- Intro. 무척 질투나는 팀을 만났다
- 1. 직감의 레이어 (떠나야 할 것 같았다 / 검정바닥: 자기결정권 / 김동률: 레전드는 레이어로 쌓인다)
- 2. 윤소정의 여행법 (시작의 본능: 낯선 도시 / 졸업작품을 제출한 도쿄 유학생 / 이건희의 직감훈련 3단계 / 이건희의 질투력)
- 3. 상하이의 편집력 (식민지는 배척하고 조계지는 흡수한다 / 상하이는 왜 크리에이티브할까 / 스티브잡스는 카피하지 않는다 오마주한다 / 새로운 오래된 것, 맥락의 편집사례)
- Outro. 이기는 습관으로 무장하며

## 핵심 일화 (3-5개)
- [장면] 블랙페이퍼 팀 만남 / 유규선 기획자 → 팀원들의 '미친 눈빛' → "아깝다, 내가 먼저 스카웃했어야 하는데!" 질투 → "앞으로 20명, 내 세계 밖 친구들에게 DM 보내겠다" 선언
- [장면] 상하이 여행 / 올드한 자신 인식 → "배우려는 태도가 불량한 상태" — 경쟁자, 롤모델, 시장 최고 팀을 적지 못했던 소정
- [장면] 트루스 비밀문서 들고 직접 피칭 → "대한민국 인재들은 다 트루스에 있네"라는 말이 들릴 때까지 뛰기로
- [장면] 러쉬 영업본부 팀과 협업 → "성수 1등 키워드 먹어보자" → 결국 키워드 1등, 1000만뷰 달성 → "이기는 감각은 자산이다"
- [장면] 피터 틸의 수직적 진보 개념 → 타자기 → 워드프로세서 = 수평 → 수직 진보 → 상하이에서 섞이는 법, 함께 더 큰 시장에서 이기는 법을 배워야 함

## 비유/프레임
- **조계지 vs 식민지**: 상하이의 역사적 공간에서 발견한 창조의 원리 — 강압으로 주입된 것(식민지)은 배척, 자발적으로 흡수한 것(조계지)은 창조로 이어짐
- **수평적 진보 vs 수직적 진보**: 타자기 100개 = 수평, 타자기 → 워드프로세서 = 수직 (피터 틸)
- **빨강머리 앤**: 75호 음악선물 테마 — 이기는 감각 + 낭만 + 유머
