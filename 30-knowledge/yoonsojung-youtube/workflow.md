# 윤소정 유튜브 영상 작업 워크플로우 (5단계)

> 영상 3편 (10만+ 검증) 직접 분석 → 진짜 공식. 다음 영상부터 이 5단계 그대로.

## 핵심 원칙

- **끝점부터 그린다** — 시청자가 *무엇을 할 사람*이 될지를 박고 역으로 스크립트
- **정의는 사용자가 박는다** — AI는 후보 펼치고 대기
- **외부 공개 숫자 가드** — 활성 구독자·매출 카피 ❌
- **카피 원칙** — 직관·실용·누구나. 시적·추상 ❌

## 5단계 워크플로우

### Step 1. 한 달 정리 — Raw Material
- 이번 한 달 가장 뜨거운 일·만남·결정 5-10개 펼치기
- 생각구독 호와 연동 가능 (호 본문 → 영상 한 편)
- 산출물: `raw.md`

### Step 2. 컨셉 박기
3가지 박기:
- **자기 폭로 한 줄** (예: "강의팔이 같다 말 들었어")
- **핵심 단어** (예: 슈퍼샘플 / 학습 괴물 / 언러닝)
- **권위 인물** (인터뷰면 누구 / 자기 정리면 권위 일화)

산출물: `concept.md`

### Step 3. 형식 결정
| 형식 | 길이 | 추천 |
|---|---|---|
| 인터뷰 | 30-47분 | 외부 권위 인물 있을 때 |
| 자기 정리 | 14분 | 자기 폭로 강할 때 |
| 혼합 | 14-20분 | 둘 다 |

### Step 4. 스크립트 작성 — 5단계 진짜 공식

```
HOOK    (0:00-0:30)  자기 폭로 1줄 + 핵심 단어 + [음악]
PROOF   (0:30-3:00)  시청자 호명 + 자기 한 달 + 권위 위임 (5개+ 자격)
FRAMES  (3:00-7:00)  3분법 (이분법·삼분법) — 4개 이상 ❌
CRY     (7:00-10:00) 자기 폭로 클라이맥스 ("그때 알았더라면…")
CALL    (10:00-끝)   오늘 한 가지 + 자료 + 비밀 라이브 + 댓글 보상
```

### Step 5. 자료·촬영 가이드
- 100페이지 노트 / 책·영화 / 비밀 라이브 / 댓글 보상
- 컷 변화 (3분마다) / 음악 [음악] 컷 / 자막 강조
- 썸네일 (정면 + 도발 한 줄)

## 시그니처 어휘 (영상 3편 공통)

- **"근데 + 진짜" 콤보** (단락마다)
- **"있거든" / "그지?" / "잖아"**
- **"여러분" / "친구들" / "얘들아"**
- **시각 비유**: 학습 괴물 / 알·피카츄 / 앨리스 / 헨젤과 그레텔 / 농사 / 가위바위보
- **권위 일화 형식**: 이름 + 자격 5개+ (회사·연도·매출·일화·평가)
- **자기 폭로 형식**: "그때 알았더라면 그렇게 ___하지 않았을 것"
