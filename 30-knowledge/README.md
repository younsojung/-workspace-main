# 30-knowledge

> **검증된 지식 아카이브.** 재사용 가능한 인사이트.

## 핵심

- **00-wiki/**: 지식이 **복리로 축적되는 위키** (Karpathy LLM Wiki). 별도 SCHEMA.md 참조.
- **37-claude-code/**: Claude Code 활용 관련 지식
- 그 외 도메인별 하위 폴더 (31-business, 32-frameworks 등)

## 00-wiki vs 일반 도메인 폴더

| | 00-wiki | 일반 도메인 (31-business 등) |
|---|---|---|
| 구조 | flat, 1파일 1토픽 | 자유 구조 |
| 형식 | SCHEMA 준수 (Infobox, 근거 인용) | 자유 |
| 용도 | AI와 함께 enriched 되는 살아있는 지식 | 자료 정리, 레퍼런스 |

**비슷한 주제가 이미 00-wiki에 있으면 그 페이지를 확장하세요.** 새 파일 금지.

## 네이밍

- 31~39 사이 숫자 사용
- 예: `31-business`, `32-business-frameworks`, `33-design`, `34-learning`
