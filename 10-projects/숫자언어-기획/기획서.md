## 학습 목표 (4 차원)

- **왜 배우는가 (Why)**: 열심히 노력하는데 성과가 제자리인 이유 — **모든 지표를 챙기려고 하면 에너지가 분산되어 임계점을 못 넘는다**. 바운더리(울타리)가 없으면 외부의 요구가 내 삶을 침범한다.
- **무엇을 배우는가 (What)**: **숫자언어 = OUTPUT 중심적 사고, 성과** = 3개월간 반드시 달성해야 하는 성과적 바운더리. = 삽질방지선.
- **어떻게 쓰는가 (How)**: 기간을 정하고 → 가지치기 → 반복. 100점 맞기 위해 20점 맞을 과목을 선택하는 의사결정.
- **무엇이 새로 생기는가 (What If)**: 80명이 동시에 자기 3개월 숫자언어 1줄을 채팅창에 게시. 3개월 후 결과를 들고 다시 만나는 공동 약속.

**전환점 (1순위) — 행동**: 끝나고 모두가 자신의 3개월 바운더리를 (1) 정하고 (2) 가지치기하고 (3) 무엇을 반복할지까지 결정한 상태로 나간다.

## 8단계 기획

### 1. 만남 — *열심히 노력하는데, 성과는 왜 제자리일까?*

> 핵심 질문: "**현재 당신이 신경 쓰고 있는 모든 숫자 지표를 리스트업해 보세요. 그중 3개월간 달성해야 할 1개만 골라야 한다면?**"

| 항목 | 내용 |
|------|------|
| **학습자 활동** | (1) 1분 침묵 — 본인이 매일/매주 신경 쓰는 숫자 지표를 다 떠올림 (예: 통장잔고, 팔로워, 몸무게, 조회수, 자본소득, 이메일 응답률, 매출, 회의 횟수…). (2) 채팅창에 본인의 **숫자 지표 리스트 5~7개**를 한 줄로 던지기. (3) "이 중 3개월간 달성할 단 1개를 골라보세요" — 5초 침묵. (4) 자기 자신에게 묻기: **"왜 못 고를까? 무엇이 중요한지 모르는 걸까?"** |
| **기획자 역할** | 동기 부여자 — 첫 도발: "**모든 지표를 챙기려고 하면 임계점을 못 넘는다.**" "**바운더리가 없으면 외부의 요구가 내 삶을 침범한다.**" — 학습자가 자기 분산 상태를 자각하게 만듦. |
| **사용 도구** | 채팅창 + 1분 침묵 타이머 |
| **시간** | 15분 (12.5%) |
| **성공 신호** | 80명이 평균 5개 이상의 지표를 떨어뜨림 = 채팅창에 400개+ 항목. 그리고 1개를 못 고르는 자기 상태를 자각함. |

---

### 2. 알아챔 — *왜 1개를 못 고르는가*

> 핵심 질문: "당신이 적은 지표들 — 이 중 어떤 것은 끌려서 적고, 어떤 것은 진짜 내 것이었나요?"

| 항목 | 내용 |
|------|------|
| **학습자 활동** | (1) 본인 리스트를 다시 봄. (2) 각 항목에 마음속으로 라벨링: 끌려간 것(타인·SNS 기준) vs 내가 정한 것. (3) Zoom 반응: "5개 이상이 *끌려간 것*인 사람" 손 들기 — 시각화. (4) 1줄 채팅: "내가 못 고른 이유는…" |
| **기획자 역할** | 안내자 — "1개를 못 고르는 건 무엇이 중요한지 모르는 게 아니라, **모든 걸 다 잡으려고 하기 때문**입니다. 그래서 임계점을 못 넘는 겁니다." |
| **사용 도구** | Zoom 반응, 채팅창 |
| **시간** | 10분 (8%) |
| **성공 신호** | "다 잡으려 한다 = 다 안 잡는 거다"라는 자각이 박힘. |

### 3. 상상 — *바운더리는 울타리다. 사과나무는 전지해야 열매가 생긴다*

> 핵심 질문: "다 자라는 게 아니라, 반드시 잘라줘야 먹을 게 생긴다 — 당신의 사과나무는 지금 어떤 모양인가요?"

| 항목 | 내용 |
|------|------|
| **학습자 활동** | (1) 정의 슬라이드 함께 읽음: **"숫자언어 = OUTPUT 중심적 사고, 성과 = 3개월간 반드시 달성해야 하는 성과적 바운더리."** (2) "바운더리가 뭐였는가? = **울타리. 삽질방지선.**" (3) 사과나무 전지 영상 30초~1분 (https://youtu.be/fjiU_GajWt8?t=102) — 다 자라는 게 아니라 잘라줘야 열매가 생기는 시각화. (4) 채팅창: "당신의 사과나무가 지금 *전지 안 된 상태*라면, 어떤 가지부터 자르고 싶나요?" 한 단어. |
| **기획자 역할** | 큐레이터 — 정의를 박고, 메타포로 *전지(가지치기)*가 핵심 액션임을 시각적으로 새김. |
| **사용 도구** | 정의 슬라이드, 사과나무 영상, 채팅창 |
| **시간** | 16분 (13%) |
| **성공 신호** | "가지치기 = 의도적 포기"라는 그림이 80명 머릿속에 박힘. 채팅에 "확장성", "다정함", "대중성" 같은 단어가 다양하게 떨어짐. |

### 4. 정보 — *정하기 / 가지치기 / 반복 — 3가지 의사결정*

> 핵심 질문: "그래서 숫자언어를 정확히 어떻게 정하는가?"

| 항목 | 내용 |
|------|------|
| **학습자 활동** | 강의 + 8분마다 빠른 채팅 반응. 다음 5파트: <br/>**① 정하는 법 3단계** — (1) 기간을 정한다 (3개월) → (2) 가지치기 한다 *(중요한 것을 고른다 = 100점 맞기 위해 20점 맞을 과목을 선택)* → (3) 반복한다. <br/>**② 사례 — 짠부** — 40만 가는 길에는 조회수가 등불, 60만 향해 갈 때는 구독자가 등불. *같은 채널인데 시기마다 숫자언어가 바뀐다.* <br/>**③ 3가지 의사결정 질문** — 1) **무엇을 포기할까?** 2) **무엇을 대충할까?** 3) **누구의 실망을 견딜 것인가?** <br/>**④ 포기의 4축** — 속도 / 대중성 / 취향 / 확장성 — 이 중 무엇을 포기할지 의도적 선택. (예: 구독자 수만 보면? *취향, 깊이있는 전문성*을 포기. "내 색깔이 왜 없어졌지?"는 그 결과.) <br/>**⑤ ROI 관점** — 같은 시간·돈·인력을 썼을 때, 가장 숫자언어가 큰 것은? 가장 적은 것은? |
| **기획자 역할** | 전문가 — 조직화해서 던지되, 매 파트 끝에 채팅 1줄 반응. "당신은 무엇을 포기할 것 같나요?" 같은 도발. |
| **사용 도구** | 슬라이드, 짠부 자료, 채팅창 |
| **시간** | 28분 (23%) |
| **성공 신호** | "100점/20점", "포기/대충/실망", "포기 4축" 세 개 framework가 80명 손에 들림. |

### 5. 연습 — *6개 페르소나 카드 — 자기 닮은 것을 고르고 모방한다*

> 핵심 질문: "이 사람들이 100점 맞기 위해 20점 맞는 건 뭘까요? — 그리고 당신은 누구를 가장 닮았나요?"

| 항목 | 내용 |
|------|------|
| **학습자 활동** | (1) 슬라이드로 6개 페르소나 카드 공개 (아래 표). (2) 본인이 가장 닮은 페르소나 1개 채팅창 투표. (3) 그 페르소나의 *100점 숫자*와 *20점 숫자*를 본인이 직접 골라봄 (워크시트 5분). (4) 채팅 공유. |
| **기획자 역할** | 코치 — 본인이 카드 1개씩 짧게 풀어주며 "이 사람은 무엇을 포기해야 할까?"를 학습자에게 묻기. |
| **사용 도구** | 페르소나 카드 슬라이드 6장, 워크시트 |
| **시간** | 18분 (15%) |
| **성공 신호** | 80명이 자기 닮은 페르소나를 고른다 + 그 페르소나의 100/20 선택 근거를 채팅에 1줄로 단다. |

**6개 페르소나 카드 (강의 자료)**

| 페르소나 | 100점 후보 | 20점 후보 |
|---|---|---|
| 대학생 | 학점 | 직무관련 프로젝트 (공모전 1등) |
| 3년차 직장인 | 업무 완수율 100% | 업무시간 단축률 30% |
| 초보 사업가 | 구독자 1만명 증가 | 매출 200% 성장 |
| 유투버 | 구독자 vs 조회수 vs 광고비 | (3개 중 1개 선택) |
| 팀장 | 브랜드 언급량 120% | 이직률 0% |
| 대표 | 영업이익 | 매출 |

*어느 게 100점이고 어느 게 20점인지는 *시기와 맥락*에 따라 다름. 정답 없음. 본인이 골라야 함.*

### 6. 나의 것 — *나의 사과나무를 전지한다* ⭐

> 핵심 질문: "당신이 100점 맞을 것은 무엇이고, 20점 맞아도 되는 것은 무엇입니까?"
> ⚡ **이 단계부터 권한이 학습자에게 이양됩니다.**

| 항목 | 내용 |
|------|------|
| **학습자 활동** | 3단계 작업: <br/>**(a)** 1단계에서 적어둔 *모든 숫자 지표*를 다시 가져옴. <br/>**(b)** **100점 맞아야 할 1개** vs **20점 맞아도 되는 것들**을 가른다. 단, 가지치기 기준은 다음 3개로 분명히 적기: <br/>　- **포기할 것** (4축 중: 속도·대중성·취향·확장성 — 무엇?) <br/>　- **대충할 것** (어디에 적게 쓸 시간·돈·에너지?) <br/>　- **실망시킬 사람** (누구의 기대를 못 충족시킬 것인가?) <br/>**(c)** 3개월 내 달성할 **숫자언어 1개**를 한 문장으로: **"나는 [영역]에서 [현재 X]에서 [3개월 후 Y]로 간다. 그러기 위해 [반복할 1가지]를 매주 한다. 그래서 [포기/대충/실망]을 감수한다."** |
| **기획자 역할** | 촉진자 — *덜* 평가하고, "왜 그 사람을 실망시킬 결심이 섰는가?" *더* 묻기. 모범 답 X. |
| **사용 도구** | 1단계 지표 리스트 + 6단계 워크시트 |
| **시간** | 18분 (15%) |
| **성공 신호** | 80개의 한 문장이 다 다르다. *포기·대충·실망* 3가지가 모두 적혀 있다 (둘만 있고 하나 빠지면 미완성). |

### 7. 다듬기 — *4가지 검증*

> 핵심 질문: "당신의 한 문장 — 정말 측정 가능하고, 진짜 포기·대충·실망 셋 다 결심됐습니까?"

| 항목 | 내용 |
|------|------|
| **학습자 활동** | (1) Zoom 소회의실 4~5명, 16~20개 방, 5분 배정. (2) 본인 한 문장 공유 → 동료가 4개 체크: ☐ 측정 가능? ☐ 3개월 내 도전적+가능? ☐ 1개로 압축됐나? ☐ **포기·대충·실망 셋 다 명시됐나?** (3) 본인 문장 1번 수정. |
| **기획자 역할** | 비판적 동료 — 본인이 80명 다 못 다루므로 *동료 피드백*에 위임. 1~2개 방 표본 점검. |
| **사용 도구** | Zoom 소회의실, 4칸 체크리스트 슬라이드 |
| **시간** | 6분 (5%) — 짧지만 *반드시* 들어감 |
| **성공 신호** | 본인 문장이 *수정*되어 다음 단계로 나옴. |

### 8. 펼치기 — *80개의 숫자언어 동시 게시 + 3개월 후 만남 약속*

> 핵심 질문: "당신의 한 문장 — 80명 앞에 던질 준비가 됐습니까?"
> 🎯 **청중**:
> 1. 같이 듣는 다른 79명 (1차 청중)
> 2. 본인이 강의 자료에 *익명 인용* (2차 청중)
> 3. **3개월 후의 자기 자신** — 다음 만남 일자 (예: YYYY-MM-DD)에 결과 들고 다시 만나기로 약속

| 항목 | 내용 |
|------|------|
| **학습자 활동** | (1) 본인 한 문장을 채팅창에 **동시에 게시** (3-2-1 카운트다운). (2) 80개 숫자언어가 한꺼번에 떨어지는 순간을 본다 — *공동 헌신의 시각화*. (3) 본인이 5~7개 임의 선정해서 짧게 읽어주며 한 줄 코멘트. (4) 마지막 약속: **"오늘부터 3개월. [다음 만남 일자] — 그때 결과를 들고 다시 만납니다. 가지치기를 잘했는지, 사과가 열렸는지 같이 봅니다."** |
| **기획자 역할** | 청중·연결자 — 본인은 한 발 물러서고, 80개의 숫자가 채팅창에 떨어지는 *현상*을 같이 본다. |
| **사용 도구** | 채팅창, 다음 만남 일자 슬라이드 |
| **시간** | 9분 (8%) |
| **성공 신호** | 채팅창에 80개 가까운 한 문장이 떨어진다. 그게 다음 사이클(3개월 후 회고)의 씨앗이 된다. |

**단계별 형성 평가 (학습 중)**
- 1단계 직후: 채팅창 메시지 수가 400개+에 가까운가? (1인당 5개 지표 × 80명)
- 4단계 직후: 채팅창 반응에 "포기", "대충", "실망" 단어들이 나오는가?
- 6단계 직후: 본인 한 문장에 **포기·대충·실망 셋 다** 적혀 있는가?
- 7단계 직후: 본인 문장이 *수정됐는가*

**최종 평가 (행동 전환점)**
- 8단계 채팅창에 본인 숫자언어 한 문장이 게시됐는가? (예/아니오 80개)
- 한 문장에 (정한 영역) + (현재→목표 숫자) + (반복할 행동) + (포기/대충/실망 명시) 4가지가 다 들어있는가?

**3개월 후 평가**
- 다음 만남 일자에 80명 중 몇 명이 *기억하고* 결과를 들고 오는가?
- 사과가 열렸는가, 가지치기가 잘 됐는가.
- 그게 다음 사이클의 만남(1단계) 재료가 된다.

### 단원 모드 권장 (선택)

이 기획은 **한 차시 120분** 압축 버전. 4단계 콘텐츠 밀도가 높아 빡빡함.
**2~4주 단원 모드**라면:
- 1차시: 만남 + 알아챔 + 상상 (지표 리스트업 → 정의 → 사과나무) — 90분
- 2차시: 정보 (3가지 의사결정 + 포기 4축 + ROI) — 90분

앤드엔 야간반이 시리즈로 운영되면 단원 모드 추천. 80명 인원 그대로 가능.
