| 항목 | 내용 |
|------|------|
| 대상 | 성인 직장인 50명. 업력 다양(추정 3~15년차). 사전 지식 = ChatGPT 들어봤지만 직접 안 써본 사람 |

## 학습 목표 (4 차원)

- **왜 배우는가 (Why)**: AI는 '아직 잘 모르겠는 도구'에서 '내 일을 한 시간 줄여주는 동료'로 인식 전환
- **무엇을 배우는가 (What)**: ChatGPT/Claude의 기본 차이 + 프롬프트 4원칙 + 일상 업무 적용 시나리오
- **어떻게 쓰는가 (How)**: 본인 업무 1개를 골라 *실제로 AI에 시키고 결과 받기*
- **무엇이 새로 생기는가 (What If)**: 내일 *자기 업무에서 AI를 한 번 쓰겠다*는 공개 약속

**전환점 (1순위)**: 🎯 **행동** — 끝나고 *반드시 한 번 적용*까지 가야 한다.

> 패턴 4 적용 (성인 학습자): 1단계 시간 상한, 4단계 전 "왜 지금"을 반복적으로 짚는다.

## 8단계 기획

### 1. 만남 — *AI 두려움을 끌어낸다* (11분, 12%)

> 핵심 질문: **"AI를 직접 써보지 않은 이유, 뭐예요?"**

| 항목 | 내용 |
|------|------|
| **학습자 활동** | (1) 짝과 1분씩 나누기 — "안 써본 이유" (2) 전체 공유 5~6개 듣기. 칠판/포스트잇에 모음. |
| **기획자 역할** | 동기 부여자 — 모든 이유 환영. 정답 X. 두려움/회의/시간 부족/실력 부족 다 OK. *공감만* 한다. |
| **사용 도구** | 화이트보드 + 포스트잇 (한 사람 한 장) |
| **시간** | 11분 |
| **성공 신호** | 모두가 한 번씩 입을 열었나? 두려움이 *언어화*됐나? 같은 이유를 가진 사람이 옆에 있다는 안도감? |

**기획자 노트**: 패턴 1 적용 — 개인 이야기 공유. 절대 "오늘은 AI를 배우겠습니다"로 시작하지 않는다.

---

### 2. 알아챔 — *우리 모두 같은 자리에 있다* (9분, 10%)

> 핵심 질문: **"이 이유들, 어떤 카테고리로 묶여요?"**

| 항목 | 내용 |
|------|------|
| **학습자 활동** | 칠판에 모인 이유들을 *학습자가 함께* 그룹핑. (예: 시간 / 실력 / 효용 회의 / 윤리 / 도구 모름) |
| **기획자 역할** | 안내자 — "이건 어디 들어갈까요?" 분류는 학습자 주도. |
| **사용 도구** | 그룹핑된 카드, 화이트보드 |
| **시간** | 9분 |
| **성공 신호** | 본인의 이유가 *나만의 것이 아니라 공통 패턴*임을 인지하는가? "아, 다들 비슷하구나"의 안도감? |

---

### 3. 상상 — *AI는 무엇 같은가* (9분, 10%)

> 핵심 질문: **"AI는 ___와 같다. 빈칸에 뭐가 들어가요?"**

| 항목 | 내용 |
|------|------|
| **학습자 활동** | 짝과 *은유* 만들기. AI = 계산기? 검색엔진? 인턴? 동료? 거울? 외국인 친구? 자기만의 비유 1개 만들기 → 짝과 공유 → 전체 3~4개 공유 |
| **기획자 역할** | 큐레이터 — 슬라이드로 다양한 비유 후보 4~5개 제시. *어느 게 맞다*는 답 없음. |
| **사용 도구** | 슬라이드 (다양한 은유 이미지 4~5장) |
| **시간** | 9분 |
| **성공 신호** | 자기 입에서 *AI를 다르게 부르는 단어*가 나오나? 추상적 두려움 → 구체적 그림으로 전환됐나? |

---

### 4. 정보 — *알고 가야 할 세 가지* (20분, 22%)

> 핵심 질문: **"오늘 가져갈 핵심 3가지"**

| 항목 | 내용 |
|------|------|
| **학습자 활동** | 강의 + 실시간 시연 듣기 — (1) ChatGPT vs Claude 차이 (2) 프롬프트 4원칙 — 역할/맥락/조건/형식 (3) 일상 업무 5가지 적용 시나리오 (이메일/회의록/리서치/문서 요약/아이디어 발산) |
| **기획자 역할** | 전문가 — 명료하게. 슬라이드 + 짧은 라이브 시연. 노트 시간 충분히. |
| **사용 도구** | 슬라이드, 노트북 (Claude 라이브 시연용) |
| **시간** | 20분 (가장 긴 단계) |
| **성공 신호** | 노트 적고 있나? *질문이 나오나*? 4원칙을 *자기 말로* 다시 말할 수 있나? |

**기획자 노트**: 성인 학습자이므로 정보 전달 *전에* "왜 지금 이걸 알아야 하는가" 한 번 더 짚고 시작.

---

### 5. 연습 — *주어진 대로 따라해본다* (16분, 18%)

> 핵심 질문: **"지금 바로 따라해보자"**

| 항목 | 내용 |
|------|------|
| **학습자 활동** | 워크시트 — 5가지 정해진 프롬프트 패턴을 *그대로* 본인 노트북에서 실행. 결과 받기. 3명씩 짝지어 결과 비교. |
| **기획자 역할** | 코치 — 돌아다니며 막힌 사람 즉각 도움. *답 알려주지 말고* "어디서 막혔어요?"로 안내. |
| **사용 도구** | 워크시트 (5개 프롬프트 템플릿), 본인 노트북, Claude/ChatGPT 계정 |
| **시간** | 16분 |
| **성공 신호** | 모두가 한 번 이상 *AI 결과물*을 받아봤나? "어, 되네!"가 한 번 이상 나왔나? |

---

### 6. 나의 것 — *내 업무에 직접 적용* ⭐ (12분, 13%)

> 핵심 질문: **"*당신의 업무*에서 AI를 적용할 *딱 한 가지*는?"**
> ⚡ **이 단계부터 기획자 권한이 학습자에게 점진적으로 이양됩니다.**

| 항목 | 내용 |
|------|------|
| **학습자 활동** | 개인 작업 — (1) 본인 업무 1개 선택 (2) AI로 어떻게 도울지 *프롬프트 1개* 직접 작성 (3) 실행 (4) 결과 평가 — 쓸만한가? |
| **기획자 역할** | 촉진자 — *답 안 줌*. "어떤 업무를 골랐어요?" "왜 그게 가장 시간이 들어요?"로만 안내. 더 가르치지 않음. |
| **사용 도구** | 본인 노트북, 본인 실제 업무 자료(이메일/문서/회의록/...) |
| **시간** | 12분 |
| **성공 신호** | 결과물이 *자기 업무에 즉시 쓸 수 있는* 수준인가? 본인 표정에서 *유레카*가 나오는가? |

**기획자 노트**: 패턴 2 적용. 5→6단계가 *권한 손바뀜* 지점. 여기서 기획자는 *덜* 가르치고 *더* 묻는다. 패턴 4 적용 — 성인 학습자에게 *자기 업무 적용*은 협상 불가능한 강제 조건.

---

### 7. 다듬기 — *짝과 함께 정제* (7분, 8%)

> 핵심 질문: **"이 프롬프트, 어떻게 더 좋아질까?"**

| 항목 | 내용 |
|------|------|
| **학습자 활동** | 3~4명 그룹 → 본인 프롬프트 + AI 결과 공유 → 서로 1~2개 피드백 → 본인 프롬프트 1차 개선 |
| **기획자 역할** | 비판적 동료 — 그룹마다 한 번씩 들름. "이 부분 더 명확하게 하려면?" 한 마디씩. |
| **사용 도구** | 종이/디지털 노트 (피드백 메모용) |
| **시간** | 7분 |
| **성공 신호** | 프롬프트가 *더 짧고 정확*해졌나? "아, 이렇게 쓰면 더 잘 나오는구나"가 발생했나? |

---

### 8. 펼치기 — *내일을 향한 한 줄 약속* (6분, 7%)

> 핵심 질문: **"*내일* 무엇을 할 것인가?"**
> 🎯 **청중**: 본인 SNS 팔로워 / 회사 단톡방 / 또는 *미래의 본인*(자기 이메일 예약 발송)

| 항목 | 내용 |
|------|------|
| **학습자 활동** | 본인 SNS/단톡방/자기 이메일에 한 줄 약속 게시 또는 발송 — "내일 [구체적 업무]에서 [구체적 AI 활용] 한 번 할 것" |
| **기획자 역할** | 청중·연결자 — 약속한 사람 손 들기 → 박수. 1주일 뒤 follow-up 이메일 보낸다 *공개 약속*. |
| **사용 도구** | 본인 디지털 채널 (SNS, 카톡, 이메일 예약 발송 도구) |
| **시간** | 6분 |
| **성공 신호** | 약속이 *공개* 또는 *미래의 자기*에게 전달됐나? 그냥 머릿속에서 끝나지 않았나? |

**기획자 노트**: 패턴 8 적용 — 청중 없는 발표는 8단계가 무너진다. *외부 청중* 강제. 1주 후 follow-up도 약속.

## 평가 / 성취 확인

> 전환점 1순위가 *행동*이므로 행동 지표 중심으로 평가.

**단계별 형성 평가**:
- 5단계 직후: 모두가 1회 이상 AI 결과 받았는가? (손 들기 확인)
- 6단계 산출물: 본인 업무에 *즉시 쓸 수 있는* 결과물 나왔는가? (그룹 공유 시 확인)

**최종 평가** (행동 영역):
- 8단계 약속의 *구체성* — "이메일에 AI 쓸게요"(추상) vs "내일 09시 클라이언트 이메일 초안을 Claude에게 시킬게요"(구체) ⭐
- 약속이 *디지털 채널에 게시*됐는지 손 들기 확인

**Follow-up** (1주일 후):
- 약속한 행동을 실행했는지 이메일/카톡 점검
- 응답률이 30% 이상이면 워크샵 성공

## 부록

### 사전 준비 체크리스트
- [ ] 1단계: 화이트보드 + 포스트잇 50장
- [ ] 3단계: 은유 슬라이드 (계산기/인턴/거울/동료/외국인 친구 등 4~5장)
- [ ] 4단계: 강의 슬라이드 + 라이브 시연 시나리오 1개 (Claude 미리 로그인)
- [ ] 5단계: 5개 프롬프트 워크시트 (인쇄 50장 또는 디지털 배포)
- [ ] 노트북 지참 사전 안내 (1주일 전)
- [ ] Claude/ChatGPT 계정 사전 발급 안내 (1주일 전)
- [ ] 8단계 follow-up 이메일 템플릿 미리 작성 (1주 후 발송)

### 사후 회고 질문
- 만남(1단계)에서 *모두가 입을 열었나*?
- 5→6단계 권한 이양이 자연스러웠나? (기획자가 6단계에서 *덜* 가르쳤나)
- 8단계 약속 게시율 / 1주 후 실행률은?
- 단원 모드(3~4주)로 운영했다면 어떻게 달랐을까?

### 미해결 / 운영 후 조정
- 50명은 6단계 개별 코칭이 어려움 — 보조 강사 2~3명 권장
- 노트북 미지참자 대비책 (옆 사람과 페어링)
- 회사 보안 정책으로 ChatGPT/Claude 접근 막힌 사람 대비 (모바일 활용)
