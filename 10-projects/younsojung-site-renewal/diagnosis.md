### 1. 본진 / 사이트의 분리
- 본진(생각구독): 월 8,000만원

- 본진이 사이트의 **3.1배**, 그러나 사이트 → 본진 동선 0
- 메인 상품(생각구독)이 사이트 메뉴 7번째에 묻힘

## 🎯 컨설턴트 권고

1. **"사이트 개편"은 부분 처방, 본질은 시스템화**
   사이트 카피 다시 쓴다고 9.8억 → 20억 가지 않음. 5,412명이 12개월 안에 100% 만료되는 *구조*를 바꿔야 함.

2. **사이트 개편을 한다면, 본진 깔때기로 재정의**
   사이트 = 단권 ebook 판매처 (X) → 사이트 = 생각구독 신규 유입 + 기존 구독자 커뮤니티 (O)

3. **2025 슬럼프의 진짜 원인 파악**
   사용자 본인 답변 (2026-05-21):
   > "25년은 인스타로만 모집했고, 재구독률이 떨어지다보니 생긴 문제. 26년은 유튜브 신규 유입 — 물을 많이 부어준 것."
   → 구조 변경 없으면 채널 의존성만 바뀜.

4. **38회·20회 재구독자 3명에게 인터뷰**
   트루스 §1.4. 이 3명의 입에서 나오는 한 문장이 *사이트 카피·상품 구성·리텐션 전략의 정답*.

5. **상품 구성 변경 = 1년+ 비중 늘리기**
   현재 1년+ 25% → 50%+로. 가격 인하 아닌 가치 재정의로.
