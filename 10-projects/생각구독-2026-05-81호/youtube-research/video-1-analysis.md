## 4. 스크립트 흐름 패턴

### 이 영상의 서사 곡선

```
[오프닝 충격 1줄]
→ "AI를 힘으로 쓰고 있냐?" — 시청자 질타형 도입

[찐 찾기]
→ "나는 공부할 때 항상 1타부터 찾아" — 방법론 신뢰 형성

[시대 정의]
→ 앨리스 배경 / 붉은 여왕 비유 — 공포 유발 (안 뛰면 퇴화)

[AI 재정의]
→ 학습 괴물 → 선생님으로 관점 전환 — 공포 → 희망으로 전환

[자기 노출]
→ "나도 처음엔 저항했어" — 시청자와 동일시

[행동 지침]
→ "지금 시작하기 절대 안 늦었어" + 헨젤과 그레텔 발자국
→ "빨리 다 같이 학생이 되자"

[심화 사례]
→ 노정석 인터뷰 직접 인용 — 권위 위임
→ 뷰티 비즈니스 + 히알로론산 (기술 신뢰도 축적)
```

### 핵심 패턴 이름: "공포 → 재정의 → 희망 → 행동"

1단계 공포: 안 움직이면 퇴화한다
2단계 재정의: AI는 적이 아니라 선생님이다
3단계 희망: 지금 시작해도 안 늦었다, 나도 그랬다
4단계 행동: 따라와. 발자국 남겨 놨다

인터뷰 클립을 권위 위임 장치로 삽입해 설득력을 높이고,
자기 노출(저항감 공개)로 시청자와 동일시를 만든다.

## 5. 공식 후보

> **"공포 1줄 → 찐 선생님 소개 → 핵심 개념 은유 → 자기 노출(나도 그랬다) → 지금 시작해도 된다 → 따라와"**

### 재현 가능한 단계형 공식

| 단계 | 역할 | 이 영상의 실례 |
|------|------|--------------|
| 1. 충격 후킹 | 시청자 즉시 질타/도발 | "AI를 힘으로 쓰고 있냐?" |
| 2. 찐 소개 | 권위 위임. "나 혼자 말이 아니야" | 노정석 — 아시아 최초 구글 엑시 |
| 3. 핵심 은유 | 개념을 이미지로 치환 | "학습 괴물" / "앨리스 배경" |
| 4. 자기 노출 | 저항감 공개 → 시청자 동일시 | "나도 처음엔 세상 빨리 안 바뀐다고 했어" |
| 5. 희망 전환 | 공포를 에너지로 바꾸기 | "지금 시작해도 절대 안 늦었어" |
| 6. 행동 지침 | 구체적 다음 단계 제시 | "헨젤과 그레텔처럼 발자국 따라와" |

### 다른 영상에도 적용될 가능성

이 공식은 이 영상에 특유한 것이 아니다.
윤소정 콘텐츠의 반복 패턴 기준으로 보면:
- 항상 "시청자를 불편하게 하는 도입부"로 시작
- 권위는 자기 주장이 아닌 찐(1타) 사례를 통해 빌려옴
- 자기 노출이 감정적 연결 핵심 장치
- 클로징은 "따라와 — 이미 발자국 남겼다" 구조

81호 슈퍼샘플에서 이 공식을 쓴다면:
충격 1줄("생각구독이 왜 필요한가") → 찐 사례 → 은유 → 나도 그랬어 → 따라와

- 자막이 자동 생성 + 1줄 연속 구조라 타임코드가 없어 정확한 시간 비율은 추정값

- 인터뷰이(노정석) 발언과 윤소정 직접 발언이 혼재해 "윤소정 공식"과 "노정석 공식"이 섞임 — 윤소정이 이를 의도적으로 자기 목소리로 재편집하는 구조

## 메인 컨텍스트 보고 (81호 작업용)

**영상 주제:** AI를 선생님으로 쓰는 학습 마인드셋 — "한 달이 1년처럼 빠른 시대에 지금 당장 학생이 되어야 한다"

**윤소정 공식:** 공포 1줄 → 찐 소개(권위 위임) → 핵심 은유 → 자기 노출(나도 그랬다) → 희망 전환 → 따라와(행동 지침)

**가장 강한 시그니처 3개:**
1. "학습 괴물" — AI를 적으로 보지 않고 선생님으로 재정의하는 핵심 은유
2. "근데" + "진짜" 반복 콤보 — 단락 전환과 강조의 디폴트 패턴
3. 자기 노출("나도 처음엔 저항했어") — 설득의 핵심 감정 장치

**81호 슈퍼샘플로 재현해야 할 핵심:**
1. **은유 하나에 집중** — "학습 괴물"처럼 한 편 전체를 꿰는 이미지 하나를 만들어야 한다. 은유 없이 개념만 나열하면 윤소정 스타일이 아님
2. **자기 노출을 먼저** — 시청자가 느낄 저항감을 윤소정이 먼저 말하는 구조. "나도 처음엔 이랬어" → "근데 이걸 알고 나서 바뀌었어" 패턴이 설득력의 핵심
