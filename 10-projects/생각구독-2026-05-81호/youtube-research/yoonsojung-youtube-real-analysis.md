## 🎨 4단계 — 윤소정 유튜브 진짜 스타일 정의

### A. 톤 — 친밀한 1인칭 직설

```
✓ "있거든", "그지?", "잖아"
✓ "여러분", "친구들", "얘들아"
✓ "근데", "진짜", "딱" — 강조 부사
✓ 짧은 문장 + 잦은 끊김
❌ 강의식 / 객관 설명조
❌ "여러분께서는…" 같은 격식체
```

### B. 자격 증명 우선 (Why before What)

```
✓ 자기 결정·경험·후회로 시작
✓ "내가 작정했다 / 이번 한 달 / 한 동료가" 같은 *구체 시작점*
❌ 이론·정의·개념부터 시작
```

### C. 권위 위임 — 외부 인물 + 자격 5개+

```
✓ 노정석 = 구글 매각 + 티스토리 + 23년 5월~ + 94편 + 비팩토리
✓ 카카오톡 투자자 = "내 자존심 긁어놓음 + 1년 시달림"
✓ 항상 *이름 + 직책 + 구체 일화 + 숫자* 함께
❌ "어떤 사람이 그러는데"
```

### D. 이분법·삼분법 구조

```
✓ 3가지로 나누기 (시대상·AI 관점·인간 역할)
✓ 또는 3가지 구분 (노력/성과, 약속/신뢰, 인정/승인)
✓ 시청자가 *자기 위치 파악* 즉시
❌ 7단계·5단계 — 너무 많으면 잊힘
```

### E. 자기 폭로 + 후회 고백 (감정 클라이맥스)

```
✓ "그때 알았더라면 그렇게 많이 울지 않았을 것"
✓ "한 동료가 강의팔이 같다고 했어"
✓ "나도 카더라 통신만 들었던 사람 중 한 명이었어"
→ 시청자 공감 + 권위 ↑
```

### F. 시각 비유 (만질 수 있는 메타퍼)

```
✓ 학습 괴물 (AI)
✓ 알·피카츄·라이추 (학습 진화)
✓ 앨리스의 붉은 여왕 (배경 속도)
✓ 헨젤과 그레텔 (발자국)
✓ 영화 3편 추천 (허·트랜센던스·루시)
→ 추상 개념을 *영화·만화·동화*로 즉시 이미지화
```

### G. 마무리 — 자료 공유 + 시청자 호출 + 시리즈

```
✓ 자료 약속: 100페이지 노트 / 영화 3편 / 책 추천
✓ "발자국 남겨놨으니 따라와" 시청자 호출
✓ 비밀 라이브 / 생각구독 깊은 진입
✓ 감성 노래 또는 시 (영상 1 엔딩)
```

### H. 핵심 단어 반복 (브랜드)

```
✓ 영상 1: "학습 괴물" / "외계 선생님" / "한 달 1년" — 각 5-10회
✓ 영상 2: "악작 같은 에너지" / "고객 승인" / "30대" — 각 5-10회
→ 한 영상 = 1-2개 핵심 단어 박힘
```
