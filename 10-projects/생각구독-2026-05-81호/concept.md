# 컨셉 (Concept)

## 메인
**슈퍼샘플의 시대**

## 부제
**배우는 시대는 끝났다. 이제는 베끼는 시대다**

## 컨셉 정의 (한 문단)
이 시대에 사람들이 따라가는 사람은, 가르치는 사람이 아니라 가장 잘 쓰는 사람이다. 사례를 모으던 시간은 끝났고, 직접 해본 한 사람이 시대의 답이 된다. 슈퍼샘플은 가르치지 않는다. 자기 일에서 가장 잘 쓰는 사람이 되어, 누군가의 따라하고 싶은 한 사람이 되는 일이다.

## 시점성 (왜 지금)
- AI가 가르치는 역할을 가져감 → 결정 상담도 AI가 더 잘함
- 그러니 사람들의 질문이 "뭘 가르쳐줄래?"에서 "누굴 따라해야 해?"로 전환
- 이림·페이스북 개발자·JP모건 친구 등 외부 사례 5월에 동시 발견
- 윤소정 본인도 "사례 모으던 강사"에서 "직접 해본 사람"으로 전환 선언

## 5월 인터뷰 raw material (10가지 사건)

### 사람 축 (8개 만남)
1. AI 트랜스포머 (미국) — 0.01% 탑 엘리트의 공유 / 진짜 병목은 안 바뀌는 인간
2. 호피 (러쉬 트레이너) — "네 시간만큼 나도 써서 네 꿈 이루겠다"
3. 동료 (강의팔이 평가) — 분노 → 증명 결심
4. 마이클잭슨 (영화) — 무지·확증편향 / 자기 노래 부르는 사람
5. 이림 (교육자) — 시대 교육자가 된 이유 = 클로드 슈퍼샘플
6. JP모건 친구 — "공부하는 사람" / 0.01% 공부 = 슈퍼샘플
7. 실리콘밸리 페이스북 개발자 — 개발자 슈퍼샘플 = 페이스북의 공식
8. AI 공부 모임 친구들 — 가장 행복했던 순간 / 본인 교육 특징 = 경험 우선
9. 안 선생님 (에너지 전문가) — 하심(下心) / 가짜 겸손은 그물에 걸린다

### 사건 축 (2개 + 본업 발견)
1. 생각마라톤 2건 — 노력·성과·바운더리 교육 / 숫자언어 = 30대 숙제
2. 안 선생님과의 대화 → "본업 = 남 잘되게 하는 것" 발견 / 강의팔이 분노의 근본 해소

### 성과 축 (4개)
1. 생각마라톤 신청자 1,000명
2. 생각구독 구독자 5,000명
3. AI 자동화 콘텐츠 20만 조회수
4. 결심: 인건비 0, 부부 둘이서 슈퍼샘플로 자립 환경 만들기

## 컨셉 도달 과정 (메타 학습 — 다음 호 봇이 참조)

### 사용자의 컨셉 뽑기 6단계
1. **5월의 사건 10가지 펼치기** (인터뷰: 사람·사건·성과 3축)
2. **핵심 개념 뽑기** (10개에서 공통점 1개)
3. **컨셉 정의 + 이미지·배경화면** (시각까지 같이)
4. 이야기 설계
5. 목차
6. 본문 채우기

### 1단계 인터뷰 질문 4종 (각 사건마다)
- 의미있던 순간은 언제였어?
- 그래서 배운 게 뭐야?
- 행동으로 성과를 낸 건 뭐야?
- 다음에 학습을 더 하고 싶은 건 뭐야?

### 부제 도달까지의 결제·후킹 룰 (사용자가 정해준 것)
- AI 단어 빼기 (시점성은 시점성대로 표현)
- 강사·30대·18년차 같은 타깃 한정 어휘 빼기 (모든 대중 포함)
- 포모(공포 후킹) 빼기 (욕망·발견·통찰 기반)
- 윤소정 80호 부제 학습 후 적용 (통념 역설·자기 폭로·신체 동사·계보·반전 발견)
- 부제 = "이 글 사고 싶다"를 만드는 한 줄
- 한 번 들었을 때 머릿속에 그림이 그려져야 함
- 카피라이터 관점 (작가 톤 아님)

### 100점 부제의 최종 장치
- **"배우다" vs "베끼다"** = 두 글자 차이로 의미 충격
- 통념 역설 (윤소정 27호 라인)
- 21자 단호
- "베끼다" 강한 동사로 그림 명확

## 사용자 자기 진단 키워드 (인터뷰 종합)
- 슈퍼샘플 = 경험을 먼저 이야기하는 사람
- 본업 = 남 잘되게 하는 것
- 하심 = 가야 할 길로 가게 하는 태도
- 부부 자립 = 슈퍼샘플의 부수 행동 결심
