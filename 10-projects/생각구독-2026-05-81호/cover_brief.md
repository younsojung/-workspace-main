# 81호 표지 디자이너 브리프

## 1. 컨셉 (절대 변경 금지)

```
[메인]   슈퍼샘플의 시대
[부제]   배우는 시대는 끝났다. 이제는 베끼는 시대다
```

### 컨셉 한 문단 (디자이너 이해용)
이 시대에 사람들이 따라가는 사람은, 가르치는 사람이 아니라 가장 잘 쓰는 사람이다. 사례를 모으던 시간은 끝났고, 직접 해본 한 사람이 시대의 답이 된다. 슈퍼샘플은 가르치지 않는다. 자기 일에서 가장 잘 쓰는 사람이 되어, 누군가의 따라하고 싶은 한 사람이 되는 일이다.

## 2. 무드 레퍼런스

**1차 레퍼런스 (핀터레스트 원본)**: `01_moodboard_pinterest.jpg`
**2차 레퍼런스 (AI 생성 시안)**: `02_moodboard_ai_generated.png`

→ 두 이미지 모두 같은 폴더에 첨부.

## 3. 비주얼 사양

### 구조
- **5-5-5-2 비대칭 격자** (총 17개)
- 작은 폴라로이드/룩북 카드 형태
- 각 카드 상단에 라벨: **"Look #01" ~ "Look #17"** (산세리프, 작게)
- 마스킹테이프로 붙인 자국 (각도·위치 일부러 살짝씩 다르게 — 손으로 붙인 느낌)

### 색감·재질
- **흑백 인물 사진** + **따뜻한 베이지 회벽 배경**
- 미세한 페인트 자국·벽 텍스처 살림
- 약간의 필름 그레인 (빈티지·아카이브 무드)

### 인물 캐스팅
- **동양인 위주** (20~40대 다양)
- 남녀 혼합
- 다양한 룩 (미니멀 스트리트웨어 ~ 에디토리얼)
- 정면 상반신 컷
- 표정 평이·차분 (강한 표정 ❌)

### 타이포
- 메인 "**슈퍼샘플의 시대**" — 큰 산세리프, 검정
- 부제 "**배우는 시대는 끝났다. 이제는 베끼는 시대다**" — 메인 아래, 작게, 검정 또는 진회색
- "Look #NN" 라벨 — 작은 산세리프, 검정 또는 진회색
- 한글 폰트는 80호 시리즈와 연속성 (Pretendard / 본명조 둘 다 가능)

### 카피 배치 옵션
- (a) 격자 위쪽 여백에 메인 + 부제 (가장 클래식)
- (b) 격자 왼쪽 또는 오른쪽 사이드바에 세로로 배치 (편집 잡지 톤)
- (c) 격자가 표지의 80% 차지, 메인은 작게 상단에만 (이미지 우선)

→ 디자이너 직감에 맡김. 단, 카피가 이미지를 가리지 않을 것.

## 4. 80호 시리즈와의 연속성

- 80호 표지 톤: 모노톤 + 시그니처 컬러 1포인트
- 81호도 같은 라인 유지 (모노톤이 기본)
- 시리즈 라벨 위치(왼쪽 상단 또는 하단)는 80호 따라감
- 윤소정 인스타·유튜브 로고 배치는 80호 그대로

## 7. 금기 사항

- ❌ AI·강사·30대·18년차 같은 단어 일체 노출 금지
- ❌ 너무 깔끔한 매끈한 벽 (거친 텍스처 유지)
- ❌ 컬러풀한 인물 사진 (반드시 흑백)
- ❌ 마스킹테이프가 너무 균일·평행하게 (손맛 살릴 것)
- ❌ 라벨 "Look #NN" 빠짐

## 8. 디자인 후 봇이 받을 파일

디자이너가 작업 끝내면 같은 폴더에 다음 파일 저장:
- `cover.png` (메인 표지 1080×1350)
- `cover_thumbnail.png` (유튜브 1280×720)
- `cover_story.png` (인스타 스토리 1080×1920)

저장되면 봇이 인지하고 Step 4 본문 작성 단계에서 표지 톤과 본문 톤 일관성 체크.
