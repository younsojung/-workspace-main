## 1. 윤소정의 컨셉 뽑기 6단계 (메타 룰)

기존 봇 4단계 → 사용자 실제 방식은 6단계:

| Step | 단계 | 핵심 룰 |
|---|---|---|
| **1** | 5월 사건 10가지 펼치기 (인터뷰) | divergent — 양껏 펼치고 |
| **2** | 핵심 개념 뽑기 | convergent — 공통점 1개 |
| **3** | 컨셉 정의 + 이미지·배경화면 | **단어 + 시각 한 묶음** |
| 4 | 이야기 설계 | (다음 호에서 학습) |
| 5 | 목차 | 챕터 3-4개 + 소제목 |
| 6 | 본문 채우기 | 채팅 → 수정 → OK → 저장 |

→ **기존 SKILL.md Step 1·2·3 = 사용자 1·2·3 대응** + **사용자 4·5 = 기존 Step 2 (목차)** 매핑.

---

## 2. 1단계 인터뷰의 4 질문 (반드시 사용)

각 사건마다:
1. **의미있던 순간이 언제였어?**
2. **그래서 배운 게 뭐야?**
3. **행동으로 성과를 낸 건 뭐야?**
4. **다음에 학습을 더 하고 싶은 건 뭐야?**

**3 분류축**: 사람 · 사건 · 성과

목표: 10개 펼친 후 → 공통점 1개

---

## 3. "10개 → 1개" 패턴은 표지에도 동일 적용

- Step 1 (인터뷰): 사건 10개 → 컨셉 1개
- Step 3 (표지): 핀터레스트 이미지 10개 → 표지 1개

→ **divergent → convergent** 사고 패턴이 윤소정 작업의 본질.

## 4. 부제 작성 룰 (81호에서 새로 발견)

### 절대 룰
- 부제 = **메인을 본 사람이 "이 글 사고 싶다"고 결심하게 만드는 한 줄**
- **한 번 들었을 때 머릿속에 그림이 그려져야** 한다
- **카피라이터 관점**으로 (작가 톤 ❌)

### 금지 어휘
- AI / 강사 / 가르치는 / 30대 / 18년차 같은 **타깃 한정 어휘**
- **포모 (공포 후킹)**: "사라진다", "망함", "가운데는 없다", "이거 모르면", "살아남는" 류

### 권장 톤
- **욕망 기반 후킹** (공포 ❌)
- **통념 역설** (윤소정 27호 라인: "A는 ~가 아니라 ~다")
- **자기 폭로 + 발견** (76호 라인)
- **신체·구체 동사** (베끼다, 따라하다, 만들다)
- **시대 vs 시대 대비** (배우는 시대 → 베끼는 시대)
- **두 글자 차이 의미 충격** (배우다 vs 베끼다)

### 80호 코퍼스에서 가장 강한 부제 라인 (참조)
- 76호: "꿈꾸지 않았다. 18년간 커리어를 설계했다" (자기 폭로 + 충격)
- 27호: "자유의 반댓말은 편협이다" (통념 역설)
- 78호: "나는 시간이 많은 어른이 되고 싶었다" (욕망 직설)
- 73호: "챗GPT는 나처럼 글을 쓴다. 하지만 화를 내진 못한다" (반전 발견)

## 6. 표지 워크플로우 (81호에서 새로 발견)

1. **사용자가 핀터레스트 무드보드 10장 모음**
2. **봇이 AI 이미지 생성 프롬프트 작성** (Midjourney/Flux/Ideogram/GPT-4o)
3. **사용자가 AI 도구로 시안 4-6장 생성**
4. **사용자가 1장 추리기**
5. **봇이 `cover_brief.md` 작성** (디자이너 사양서)
6. **외부 디자이너 작업** → `cover.png` 폴더에 저장
7. **봇이 표지 인지** + 본문 톤 일관성 체크

→ 기존 봇 Step 3 (표지) 워크플로우 전면 재설계.
